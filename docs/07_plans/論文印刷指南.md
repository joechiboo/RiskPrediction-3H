## 一、印刷方式：單面印刷

> **學校規定**：80 頁以上雙面印刷；80 頁以下單面印刷。

目前完整論文約 60+ 頁，採 **單面印刷**。各項起始頁不限在奇數頁，直接連續即可。

## 二、檔案結構與裝訂順序

### 獨立檔案（不合併進主文件）

| 項目 | 檔案 | 說明 |
|------|------|------|
| 封面（含書背） | 印刷店製作 | 天空藍 200 磅西卡紙/雲彩紙，交給印刷店排版 |
| 書名頁 | `00_封面書名頁.docx` | 內頁格式的封面，**獨立 Word 檔**，不編頁碼 |
| 授權書 | 論文系統列印 | 口試後從論文系統產生，親簽，A4 原稿勿縮放 |
| 審定書 | 助教懶人包套用檔 | 口試當天口委簽名，掃描裝入 |

> **重點**：書名頁是獨立檔案，不要合併到論文主文件裡。
> 印刷時按順序排好頁面即可。

### 論文主文件（合併成一個 Word / PDF）

| 順序 | 內容 | 頁碼格式 | 來源 |
|:----:|------|:--------:|------|
| 1 | 謝辭 | 不編碼 或 i | `00a_謝辭.md` |
| 2 | 中文摘要 | i (或 ii) | `00b_中文摘要.md` |
| 3 | 英文摘要 | ii (或 iii) | `00b_英文摘要.md` |
| 4 | 目錄 | iii~ | Word 自動生成 |
| 5 | 表目錄 | 接續 | Word 自動生成 |
| 6 | 圖目錄 | 接續 | Word 自動生成 |
| 7 | 第一章～第五章 | 1, 2, 3... | `01_`~`05_` 各章 |
| 8 | 參考文獻 | 接續 | `06_參考文獻.md` |

### 紙本裝訂順序（從前到後）

```
封面（天空藍西卡紙）
  ↓
書名頁（獨立檔案，白紙）
  ↓
授權書（口試後，親簽）
  ↓
審定書（口試後，口委簽名）
  ↓
論文主文件（謝辭 → 摘要 → 目錄 → 正文 → 參考文獻）
  ↓
封底（天空藍西卡紙，空白）
```

## 三、頁碼設定

論文主文件需要 **兩套頁碼系統**：

| 區段 | 頁碼格式 | 範圍 |
|------|:--------:|------|
| 前置頁面 | 小寫羅馬數字 i, ii, iii... | 謝辭 → 圖目錄 |
| 正文＋後置 | 阿拉伯數字 1, 2, 3... | 第一章 → 參考文獻 |

### Word 設定步驟

1. 在「圖目錄」最後一頁末尾插入 **「下一頁」分節符號**
2. 在正文第一頁：
   - 取消「連結到前一節」
   - 設定頁碼格式 → 起始頁碼 = 1
3. 前置頁面區段：
   - 設定頁碼格式 → 羅馬數字 (i, ii, iii)
4. 書名頁因為是獨立檔案，不需要在主文件中處理

## 三之一、排版工作流程（數位準備）

> 印刷前需完成以下五步，依序執行。

### Step 1：各章 md → docx 轉換 + 格式微調

將各章 Markdown 透過 pandoc 轉成 Word，再手動微調格式：

```bash
cd d:\Personal\Project\RiskPrediction-3H\docs\00_thesis\chapters
pandoc "01_第一章_緒論.md" -o "01_第一章_緒論.docx" --reference-doc="03_第三章_研究方法.docx"
# 其餘章節同理
```

轉換後微調項目：

- [ ] 刪除 pandoc 產生的 `---` 水平線
- [ ] 中文字型 → 標楷體 12pt / 英文 → Times New Roman 12pt
- [ ] 行距 → 1.5 倍行高
- [ ] 數學公式手動檢查（pandoc 轉換後可能跑掉）
- [ ] 表格內粗體數值改回一般格式

| 章節 | 轉換狀態 |
|------| :------: |
| 00a 謝辭 | 待轉換 |
| 00b 中文摘要 | 待轉換 |
| 00b 英文摘要 | 待轉換 |
| 01 第一章 緒論 | ✅ 已有 docx |
| 02 第二章 文獻探討 | ✅ 已有 docx |
| 03 第三章 研究方法 | ✅ 已有 docx |
| 04 第四章 實驗結果 | ✅ 已有 docx |
| 05 第五章 結論 | ✅ 已有 docx |
| 06 參考文獻 | ✅ 已轉換 |

### Step 2：合併所有章節為單一 Word 主文件

合併順序（依論文結構）：

```text
謝辭 → 中文摘要 → 英文摘要 → [目錄預留位置] → 第一章 → ... → 第五章 → 參考文獻
```

合併方式：在 Word 中使用 **插入 → 物件 → 文字檔案** 依序插入各章節 docx，或手動複製貼上。

> **注意**：書名頁是獨立檔案，**不要**合併進主文件。

### Step 3：分節符號 + 頁碼設定（羅馬/阿拉伯）

1. 在「圖目錄」最後一頁末尾插入 **「下一頁」分節符號**
2. 前置頁面（謝辭 → 圖目錄）：羅馬數字 i, ii, iii...
3. 正文（第一章 → 參考文獻）：阿拉伯數字 1, 2, 3...，起始頁碼 = 1
4. 正文第一頁：取消「連結到前一節」

> 詳細 Word 操作見下方「三、頁碼設定」。

### Step 4：自動目錄 + 圖表目錄生成

1. 在預留位置插入 **目錄**（參考 → 目錄 → 自訂目錄，顯示 3 層級）
2. 插入 **表目錄**（參考 → 插入圖表目錄，標籤 = 「表」）
3. 插入 **圖目錄**（參考 → 插入圖表目錄，標籤 = 「圖」）

> **目錄必須最後才生成**，因為頁碼在 Step 3 才確定。

### Step 5：最終排版檢查（圖表不跨頁、頁碼對齊）

- [ ] 圖表不跨頁（段落 → 分行與分頁 → 與下段同頁 + 段落中不分頁）
- [ ] 目錄頁碼與實際一致（全選 Ctrl+A → 更新功能變數 F9）
- [ ] 圖表編號連續、無遺漏
- [ ] 邊界：上 3.5cm / 下 2.5cm / 左 3cm / 右 2.5cm
- [ ] 頁碼位置：底端置中
- [ ] 試印一份確認排版無誤

## 四、實際印刷流程

### Step 1：公司自行列印（單面）

列印以下內容，按順序疊好：

| 順序 | 內容 | 來源 | 備註 |
| :----: | ------ | ------ | ------ |
| 1 | 書名頁 | `00_封面書名頁.docx` | 獨立檔案，1 頁 |
| 2 | 授權書 | 論文系統列印 | 口試後才有 |
| 3 | 審定書 | 助教套用檔 | 口試後才有 |
| 4 | 論文主文件 | 合併的 Word/PDF | 謝辭 → 參考文獻，全部 |

> 初稿階段（口試前）：只印 1 + 4，授權書和審定書的位置先留空。

### Step 2：拿去印刷店

交給印刷店的東西：

- 印好的內頁（整疊）
- 封面資訊（校名、系所、論文題目、研究生姓名、指導教授、中華民國 115 年 7 月）

印刷店負責：

- 製作封面 + 書背 + 封底（天空藍 200 磅西卡紙/雲彩紙）
- 平裝上光裝訂

### 印刷規格

| 項目 | 規格 |
|------|------|
| 紙張 | A4 單面 |
| 裝訂 | 平裝上光 |
| 封面材質 | 天空藍色 200 磅銅西卡紙或雲彩紙 |
| 繳交冊數 | **3 冊**（圖書館離校用） |
| 自留 | 建議多印 1-2 冊（自己＋教授） |

## 五、圖表排版原則

> 參考手邊紙本碩論風格：圖表獨立呈現，不硬擠在段落中間。

### 表格

- **獨立放置**：表格上方為標題（如「表 4-1 八種模型三項疾病預測結果」），下方接續正文
- **大表獨佔一頁**：欄位多或列數多的表（如表 4-1 八模型比較、表 4-10 類別不平衡比較）直接佔滿一頁
- **不跨頁**：若表格放不下，先試著縮小字級（最小 10pt），真的太長再拆分為上下兩表
- **表格內字型**：中文標楷體 / 英文 Times New Roman，字級可比正文小（10-11pt）

### 圖片

- **置中放大**：佔頁面寬度 80-100%，確保圖中文字可辨識
- **圖說在下方**：格式為「圖 X-X 圖片說明」，置中
- **解析度**：300 dpi 以上，不用螢幕截圖
- **大圖獨佔一頁**：如 ROC 曲線（圖 4-1）、SHAP Beeswarm（圖 4-3）、健檢次數趨勢（圖 4-7）

### 預估頁數影響

目前 pandoc 緊湊排版約 46 頁正文。改為圖表獨立頁風格後：

- 33 個圖表（Ch3: 3 圖 8 表 + Ch4: 7 圖 15 表），每個佔半頁到一頁
- 正文預估撐到 **60-70 頁**，符合碩論常見篇幅

## 六、撰寫格式補充（參考同學版指引）

> 來源：`21_資訊科學領域論文格式要點.docx`，以學校規定為主，以下為補充參考。

### 標題層次

| 層級 | 字級 | 位置 | 字形 |
|------|:----:|------|------|
| 章標題（第一章） | 18pt | 置中 | 加黑 |
| 節標題（1.1） | 14pt | 置中 | 加黑 |
| 次標題（1.1.1） | 12pt | 靠左 | 加黑 |

- 字體大小依章→節→次標題由大變小，粗細由粗變細
- 一段至少三行，三段以上才構成一個次標題，一節至少三個次標題
- 章節標題若落在該頁最後一行，應移至下頁

### 方程式格式

- 方程式編號右對齊，如 `(1)`
- 方程式中所有符號需在方程式出現前或緊接其後定義
- 引用時直接寫 `(1)`；句首才寫 `方程式(1)`

### 圖表續頁處理

- 同一圖（表）必須在同一頁，不可分頁
- 若圖表超過一頁：第二頁標「(續)」或「(續1)」，第三頁「(續2)」
- 超過 5 頁的圖表放附錄
- 圖（表）與主文前後各空一行

### 外國語規則

- 首次出現的外國語名詞或縮寫，須翻譯成中文並將原文以括號置於翻譯之後
- 專有名詞英文首字母大寫，其餘小寫（外國人名不須譯成中文）
- 縮寫每個字母大寫，全文在先、縮寫附於其後
- 範例：頻率-時間分析 (Frequency-Time Analysis, FTAN)

### 摘要結構（500-800 字）

1. 問題的陳述
2. 解決該問題所用研究方法及簡述
3. 重要改善成果或創新發現

> 摘要中不可含歷史背景、前言、老舊資訊、標準方法細節、未來構想、原始數據。

- [ ] 全文錯別字檢查
- [ ] 圖表編號連續、無遺漏
- [ ] 圖表不跨頁（完整呈現於同一頁）
- [ ] 參考文獻格式統一

- [ ] 目錄頁碼與實際一致（更新功能變數）

### 格式確認

- [ ] 章標題：18pt 置中加黑
- [ ] 節標題：14pt 置中加黑
- [ ] 節以下次標題：12pt 靠左加黑
- [ ] 正文中文字型：標楷體 12pt
- [ ] 正文英文字型：Times New Roman 12pt
- [ ] 行距：1.5 倍行高
- [ ] 邊界：上 3.5cm / 下 2.5cm / 左 3cm（裝訂側）/ 右 2.5cm
- [ ] 頁碼位置：底端置中
- [ ] 各章開頭另起新頁

### 檔案確認

- [ ] 書名頁為獨立 Word 檔（`00_封面書名頁.docx`）
- [ ] 論文主文件為單一 Word/PDF

## 八、常見踩坑紀錄

| 問題 | 解法 |
|------|------|
| 書名頁被合併進主文件 | 書名頁必須是獨立檔案，印刷時單獨夾在封面後面 |
| Word 目錄頁碼錯誤 | 印刷前務必「全選 → 更新功能變數」(Ctrl+A → F9) |
| 圖片模糊 | 確保圖片為 300 dpi 以上，不要用螢幕截圖 |
| 表格跨頁 | 段落設定 → 與下段同頁 + 段落中不分頁 |
| 浮水印 | 電子檔需加學校浮水印（圖書館 4F 有 Adobe Acrobat 可用） |
| 授權書比例跑掉 | 掃描時 A4 原稿，勿縮放 |
