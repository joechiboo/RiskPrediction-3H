### 4. 確認訓練/測試比例

**問題**：我們的比例分配是 80/20 嗎？

**答案**：是。5-Fold CV = 80% 訓練 / 20% 測試，輪 5 次。

**補充細節**：
- 使用 StratifiedGroupKFold（非普通 KFold）
- Group = 受試者 ID → 同一人的所有滑動窗口紀錄不拆到不同折（防 data leakage）
- Stratified → 每折的疾病陽性比例一致
- 論文報告的 AUC 為 5 折測試集平均
