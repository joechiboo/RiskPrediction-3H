### 研究方向

1. 整理 [02_literature/](../02_literature/) 各篇論文的：
   - 期刊名 + IF + Quartile
   - 主題（單疾病 / 多疾病 / Δ / 縱向）
   - 方法（LR / DL / 符號回歸）
2. **分類分布**：哪些「方法+主題」組合最常出現在哪些期刊
3. **本研究的「目標期刊匹配度」**：
   - Diagnostics 最常收哪類論文？
   - IEEE Access 跟我們最像的是哪幾篇？
   - IJMI 的近期收稿 trend？

- `docs/02_literature/summaries/期刊分類_投稿目標分析.md`
- 一張 「方法 × 主題 × 期刊」三軸矩陣
- **明確的「投稿優先序」建議**
