## 二、研究動機 flow

### 教授指示

研究敘事流程：**文獻 → 缺口 → 動機 → 研究目標**

### 現況分析

- Ch1：研究背景 → 研究目標 → 研究貢獻
- Ch2：相關研究 → 三高預測 → 縱向資料 → 類別不平衡 → 研究缺口 → 研究動機

**小問題**：Ch1 §2 直接講研究目標時，讀者還沒看到 Ch2 的文獻和缺口，邏輯上有斷裂。

### 建議調整（小修正，避免大改架構）

1. **Ch1 §1 結尾加橋接段** — 「以下章節將先回顧相關研究（第二章），識別研究缺口，進而說明本研究的具體目標」
2. **Ch1 §2 主要目標前加引導句** — 「基於上述背景與第二章將討論的研究缺口（§2.5），本研究的主要目標是...」
3. **Ch2 §5.3 研究動機補一句** — 「上述缺口共同構成本研究的核心動機，呼應 §1.2 之主要目標」

**理由**：
- 不大改架構，避免動到目次、頁碼、交叉引用
- 透過橋接句讓讀者感受到 flow
- Ch1 目標與 Ch2 缺口本來就對應（縱向資料、Δ 特徵、可解釋性），只是沒明說
