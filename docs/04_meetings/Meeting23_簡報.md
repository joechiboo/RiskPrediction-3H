## Slide 3: 新增實驗 — §6.7.3 各時間點獨立預測力

### 實驗設計

- **對象**：2,526 名有完整 5 次健檢的參與者
- **方法**：分別使用 T1、T2、T3、T4 各自獨立預測 T5
- **特徵**：10 個（sex + Age + 8 項生物標記）
- **評估**：LR + StratifiedKFold 5-fold CV

### 結果（AUC）

| 時間點 | 距 T5 | 高血壓 | 高血糖 | 高血脂 |
|:------:|:-----:|:------:|:------:|:------:|
| T1 | ~8 年 | **0.786** ⬆ | 0.868 | 0.814 |
| T2 | ~6 年 | 0.627 | 0.892 | 0.843 |
| T3 | ~4 年 | 0.666 | 0.912 | 0.853 |
| T4 | ~2 年 | 0.651 | **0.931** ⬆ | **0.867** ⬆ |

### 關鍵發現

- **高血糖 / 高血脂**：T4（最近）最強，隨距離遞減 → 符合直覺 ✅
- **高血壓**：T1（最遠）反而最強，高出 T2-T4 達 **0.12-0.16** → 反直覺 ❓

> 教授上次的直覺：「理論上越遠應越不準」— 高血糖和高血脂確實如此，但高血壓例外。

## Slide 4: T1 假說 — 為什麼最早時間點預測力最強？（§7.5.2）

### 推測原因：入組篩選 + 降壓藥物干預

```
T1（入組）        T2          T3          T4          T5（預測目標）
  │                │           │           │              │
  ├─ 篩選排除      ├─ 未篩選    ├─ 未篩選    ├─ 未篩選      │
  │  已確診者       │           │           │              │
  │                │           │           │              │
  ├─ 血壓 =       ├─ 血壓可能  ├─ 同左      ├─ 同左        │
  │  自然基線       │  被藥物    │           │              │
  │  信號純淨 ✅    │  壓低 ⚠️   │           │              │
```

### 三步推論

1. **T1 信號純淨**：Luo et al. 在 T1 排除了已確診高血壓者（含服藥者），因此 T1 血壓值反映自然基線狀態
2. **T2-T4 受藥物汙染**：追蹤期間部分受試者開始服用降壓藥，導致血壓值被壓低，但潛在風險仍在
3. **高血糖/高血脂未受影響**：降壓藥見效快（數日至數週），降血糖/降血脂藥效果較緩慢，對量測值的短期干擾較小

### 限制

> 本資料集**不含用藥紀錄**，無法直接驗證此假說。列入 §8.2 研究限制，建議未來研究在有用藥紀錄的資料集上驗證。

---

**📝 講稿（1.5分鐘）**

> 我的假說是：T1 是入組時間點，Luo 的篩選排除了 T1 已確診高血壓的人，包括吃降壓藥的人。所以 T1 的血壓值是自然基線狀態，信號比較純淨。
>
> 但 T2 到 T4 沒有再做篩選，有些人中間開始吃降壓藥了，血壓被藥物壓下來，可是他們其實還是高風險。模型看到的是被藥物壓低的數值，所以區分能力變差。
>
> 高血糖和高血脂沒有這個問題，因為降壓藥的效果很快——幾天到幾週就能明顯降低血壓；但降血糖和降血脂藥的效果相對慢，對量測值的干擾比較小。
>
> 不過我們資料集沒有用藥紀錄，沒辦法直接驗證，所以這個寫在研究限制裡面，作為未來研究的方向。
