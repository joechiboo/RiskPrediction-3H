## 修正摘要

### 十、研究生申請碩士學位考試，應依下列規定辦理：

（一）申請時，填具申請書並檢齊下列各項文件：

1. 臺灣學術倫理教育資源中心核發之六小時（含）以上修課證明。
2. 歷年成績表一份。
3. 論文初稿或代替碩士論文之作品、成就證明連同書面報告、技術報告或專業實務報告，及其摘要各一份，**且提供經指導教授簽名確認之論文原創性比對結果證明文件。**

### 十六、學位考試舉行後，各系、所、學位學程應俟研究生繳交修訂完成且內附審定書之論文或代替論文之作品、成就證明連同書面報告、技術報告或專業實務報告**及經指導教授簽名確認已修訂完成後論文之原創性比對結果證明文件**，始得將合格之學位考試成績送進修推廣處進修教育中心登錄。

取得碩士學位者應將其取得學位之論文、書面報告、技術報告或專業實務報告及其電子檔，送本校圖書館及國家圖書館保存之，並應提供公眾於國家圖書館內閱覽紙本，或透過獨立設備讀取電子資料檔；同意經依著作權法規定授權重製、透過網路於館內或館外公開傳輸，或其他涉及著作權之行為者，於畢業離校時，應將親筆簽名之**本校圖書館及國家圖書館學位論文授權書**，交由本校圖書館，並由本校圖書館遞送國家圖書館保存之。**學位論文、書面報告、技術報告或專業實務報告一經繳交保存，即不得進行抽換，惟若涉及學術倫理案件或經系所會議認定之重大事件須抽換或撤銷時，不在此限。**

## 論文原創性比對證明文件

本系規範同學申請學位考試時、學位考試後（離校手續前）共**兩次**的論文原創性比對結果證明文件如下：

### 比對完成後請自行印出：

```
「電子回條」 + 「【目前檢視】/擷取有百分比的 pdf 頁面」
```

共 2 頁紙本由學生本人於紙本空白處**簽名 + 簽日期**後再送指導老師確認後於紙本空白處**簽名 + 簽日期**。

### 離校程序：

且於辦理離校程序時會將學位考試後（離校手續前）之碩士學位論文原創性比對報告完整電子檔儲存於**離校光碟片**中。

---

## 重點整理

| 時間點 | 需要文件 |
|--------|----------|
| **申請學位考試時** | 論文原創性比對證明（指導教授簽名） |
| **學位考試後（離校前）** | 修訂完成後論文原創性比對證明（指導教授簽名） |

### 比對證明文件格式

- 電子回條（1 頁）
- 目前檢視 / 有百分比的 PDF 頁面（1 頁）
- 學生簽名 + 日期
- 指導教授簽名 + 日期
