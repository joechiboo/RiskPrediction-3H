# 美國膳食指南 2025-2030 深度解析

> **文件來源**：Dietary Guidelines for Americans, 2025-2030
> **發布日期**：2026 年 1 月 7 日
> **發布單位**：USDA + HHS
> **官方網站**：[realfood.gov](https://realfood.gov)
> **解析日期**：2026-01-12

---

## 一、歷史性變革概述

### 1.1 核心訊息

> **"The message is simple: eat real food."**

這版指南被稱為「美國聯邦營養政策史上最重大的重置」，由 HHS 部長 Robert F. Kennedy Jr. 和 USDA 部長 Brooke Rollins 共同發布。

### 1.2 背景：美國健康危機

| 統計數據 | 說明 |
|---------|------|
| 90% | 醫療支出用於治療慢性病 |
| 70%+ | 成年人過重或肥胖 |
| 1/3 | 12-17 歲青少年有前糖尿病 |

指南明確指出：這些疾病**不是基因命運**，而是「標準美國飲食」(Standard American Diet) 的可預測結果。

## 二、每日攝取建議總覽（2,000 卡路里基準）

| 食物類別 | 每日份量 | 重點說明 |
|---------|---------|---------|
| **蛋白質** | 1.2-1.6 g/kg 體重 | 每餐都要吃蛋白質 |
| **乳製品** | 3 份 | **全脂**，無添加糖 |
| **蔬菜** | 3 份 | 多樣化、多顏色 |
| **水果** | 2 份 | 原型為主 |
| **全穀物** | 2-4 份 | 高纖維 |
| **添加糖** | **0 g（理想）** | 單餐不超過 10g |
| **鈉** | < 2,300 mg | 14 歲以上 |
| **飽和脂肪** | < 10% 總熱量 | |

## 三、六大核心建議詳解

### 3.1 每餐優先攝取蛋白質 (Prioritize Protein)

**推薦來源**：
- 動物性：蛋、雞肉、海鮮、紅肉
- 植物性：豆類、扁豆、堅果、種子、大豆

**烹調建議**：
- ✅ 烘烤、燒烤、炙烤、炒
- ❌ 避免油炸

**目標**：1.2-1.6 g/kg 體重/天

### 3.2 攝取乳製品 (Consume Dairy)

**重大變化**：推薦**全脂乳製品**（而非過去的低脂/脫脂）

- 3 份/天
- 無添加糖
- 是蛋白質、健康脂肪、維生素、礦物質的優質來源

### 3.3 全天攝取蔬果 (Vegetables & Fruits)

- 蔬菜 3 份 + 水果 2 份
- 優先選擇**原型**食物
- 冷凍、乾燥、罐頭也可（無添加糖）
- 100% 果汁應限量或稀釋

### 3.4 納入健康脂肪 (Healthy Fats)

**推薦來源**：
- 肉類、雞肉、蛋
- 富含 Omega-3 的海鮮
- 堅果、種子
- 全脂乳製品
- 橄欖、酪梨

**烹調用油優先順序**：
1. 橄欖油（含必需脂肪酸）
2. 奶油
3. 牛油 (beef tallow)

**注意**：飽和脂肪 < 10% 總熱量

### 3.5 專注全穀物 (Whole Grains)

- 2-4 份/天
- 優先高纖維全穀物

**應大幅減少**：
- 白麵包
- 即食/包裝早餐
- 麵粉墨西哥餅
- 餅乾

### 3.6 限制超加工食品與添加糖 ⭐⭐⭐

這是本版指南**最具突破性**的建議：

> **"No amount of added sugars or non-nutritive sweeteners is recommended or considered part of a healthy or nutritious diet."**
>
> （無論是添加糖或非營養性甜味劑，任何攝取量都不被建議或視為健康飲食的一部分）

**應避免**：
- 高度加工的包裝食品
- 含人工香料、石油基色素、人工防腐劑的食品
- 含糖飲料（汽水、果汁飲料、能量飲料）
- 低卡非營養性甜味劑

**添加糖限制**：
| 情境 | 上限 |
|------|------|
| 單餐 | ≤ 10g |
| 穀物零食（如餅乾）| ≤ 5g / ¾ oz |
| 乳製品零食（如優格）| ≤ 2.5g / ⅔ cup |

**常見添加糖名稱**：
- 高果糖玉米糖漿、龍舌蘭糖漿、玉米糖漿
- 果糖、葡萄糖、右旋糖、蔗糖
- 蔗糖、甜菜糖、粗糖
- 麥芽糖、乳糖
- 濃縮果汁、蜂蜜、糖蜜

**非營養性甜味劑**（也應限制）：
- 阿斯巴甜 (aspartame)
- 蔗糖素 (sucralose)
- 糖精 (saccharin)
- 木糖醇 (xylitol)
- 醋磺內酯鉀 (acesulfame K)

## 四、鈉與酒精

### 4.1 鈉攝取建議

| 年齡 | 每日上限 |
|------|---------|
| 1-3 歲 | < 1,200 mg |
| 4-8 歲 | < 1,500 mg |
| 9-13 歲 | < 1,800 mg |
| 14 歲以上 | < 2,300 mg |

**注意**：高度活動者可能需要更多鈉來補充汗液流失。

### 4.2 酒精

- 減少攝取對整體健康更好
- **完全禁止**：孕婦、酒精成癮康復者、服用特定藥物者
- 有家族酒精成癮史者應特別注意

## 五、特殊族群建議

### 5.1 嬰幼兒（0-4 歲）

| 階段 | 建議 |
|------|------|
| 0-6 個月 | 純母乳（或鐵強化配方奶）|
| 6 個月起 | 開始副食品，持續哺乳 |
| 12 個月 | 停配方奶，改全脂牛奶 |

**重要**：
- 維生素 D 補充：400 IU/天（出生後不久開始）
- **完全避免添加糖**
- 6 個月左右引入過敏性食物（花生、蛋、貝類、小麥）

**副食品推薦**：
- 肉類、雞肉、海鮮
- 蔬菜、水果
- 全脂優格、起司
- 全穀物
- 豆類、堅果/種子（適合嬰兒的形式）

### 5.2 兒童（5-10 歲）

- 專注原型、高營養密度食物
- **全脂乳製品**對大腦發育和能量需求重要
- 避免含咖啡因飲料
- **不建議任何添加糖**
- 讓烹飪成為家庭樂趣

### 5.3 青少年（11-18 歲）

- 快速生長期，需要更多能量、蛋白質、鈣、鐵
- 女生因月經需特別注意鐵質
- 鈣和維生素 D 對達到骨質峰值至關重要
- **大幅限制**含糖飲料和能量飲料
- 鼓勵參與採購和烹飪

### 5.4 年輕成人

- 大腦持續發育
- 優化骨骼健康達到峰值
- 支持生殖健康：
  - 女性：健康脂肪、鐵、葉酸
  - 男性：健康脂肪、蛋白質

### 5.5 孕婦

**優先營養素**：鐵、葉酸、碘

**推薦食物**：
- 富含鐵的肉類
- 富含葉酸的綠葉蔬菜和豆類
- 富含膽鹼的蛋
- 富含鈣的乳製品
- 低汞、富含 Omega-3 的海鮮（鮭魚、沙丁魚、鱒魚）

**建議諮詢醫療專業人員關於產前維生素**

### 5.6 哺乳期婦女

需要更多能量和營養素：
- 維生素 B12（肉類、雞肉、蛋、乳製品）
- Omega-3（海鮮）
- 葉酸（豆類）
- 維生素 A（蔬菜）

### 5.7 老年人

- 可能需要較少熱量，但需要**同等或更多**關鍵營養素
- 優先營養素：蛋白質、維生素 B12、維生素 D、鈣
- 當飲食攝取不足時，可能需要強化食品或補充劑

### 5.8 慢性病患者 ⭐

> 遵循膳食指南可幫助**預防發病或減緩慢性病進展**，特別是心血管疾病、肥胖和第二型糖尿病。

**重要建議**：
- 某些慢性病患者採用**低碳水化合物飲食**可能有更好的健康結果
- 應與醫療專業人員討論適合個人的飲食方式

### 5.9 素食者與純素者

**注意事項**：
- 素食者常缺乏：維生素 D、E、膽鹼、鐵
- 純素者缺乏更廣泛：維生素 A、D、E、B6、B12、核黃素、菸酸、膽鹼、鈣、鐵、鎂、磷、鉀、鋅、蛋白質

**建議**：
- 定期監測營養狀態（特別是鐵、B12、維生素 D、鈣、碘）
- 針對性補充
- 多樣化植物蛋白來源
- 透過食物製備技術提高礦物質生物利用度

## 六、腸道健康 (Gut Health)

指南特別提到：

- 腸道含有**數兆細菌**和其他微生物（微生物組）
- 健康飲食支持平衡的微生物組和健康消化
- **超加工食品會破壞這種平衡**
- 支持多樣化微生物組的食物：
  - 蔬菜、水果
  - 發酵食品（德國酸菜、泡菜、克菲爾、味噌）
  - 高纖維食物

## 七、與過去版本的重大差異

| 項目 | 2020-2025 版 | 2025-2030 版（本版）|
|------|-------------|-------------------|
| **乳製品** | 低脂/脫脂優先 | **全脂**推薦 |
| **添加糖** | < 10% 熱量 | **0 是理想值** |
| **超加工食品** | 未特別強調 | **明確反對** |
| **蛋白質優先** | 中性 | **每餐優先** |
| **動物性蛋白** | 與植物性並列 | **優先列出** |
| **非營養性甜味劑** | 未明確限制 | **不建議** |
| **奶油/牛油** | 不推薦 | **可作為烹調用油** |
| **低碳飲食** | 未提及 | **可用於慢性病** |

## 八、爭議與批評

### 8.1 哈佛公衛學院的批評

> 「與諮詢委員會報告的建議存在**政治驅動的偏離**，缺乏透明度或科學依據」

主要擔憂：
- 過度強調動物性蛋白
- 弱化植物性選項
- 全脂乳製品推廣（飽和脂肪疑慮）
- 植物油被忽略

### 8.2 美國心臟協會 (AHA) 的擔憂

- 鹽調味和紅肉建議可能導致鈉和飽和脂肪超標
- 仍建議低脂/脫脂乳製品對心臟健康更好
- 呼籲更多蛋白質攝取量和來源的研究
- 鼓勵植物性蛋白、海鮮和瘦肉

## 九、與三高預測研究的關聯

本指南對我們的研究有以下啟示：

### 9.1 飲食與三高的直接關聯

指南明確指出飲食可幫助預防或減緩：
- 心血管疾病（高血壓）
- 第二型糖尿病（高血糖）
- 肥胖（三高風險因子）

### 9.2 可能的研究延伸

| 研究方向 | 說明 |
|---------|------|
| 飲食模式與三高風險 | 超加工食品攝取與發病率 |
| 低碳飲食介入 | 對已有三高者的效果 |
| 添加糖與代謝症候群 | 糖攝取量與三高關聯 |

### 9.3 預測模型特徵考量

未來若有飲食資料，可考慮納入：
- 超加工食品攝取頻率
- 添加糖攝取量
- 蛋白質攝取來源
- 飽和脂肪比例

## 十、總結

### 10.1 核心精神

**"Eat Real Food"**（吃真正的食物）

### 10.2 三大行動重點

1. **增加**：蛋白質、全脂乳製品、蔬果、健康脂肪、全穀物
2. **減少**：超加工食品、添加糖、精製碳水化合物
3. **避免**：含糖飲料、人工添加物、非營養性甜味劑

### 10.3 每日份量速查表（2,000 卡）

| 類別 | 份量 |
|------|------|
| 蛋白質 | 1.2-1.6 g/kg |
| 乳製品 | 3 份 |
| 蔬菜 | 3 份 |
| 水果 | 2 份 |
| 全穀物 | 2-4 份 |
| 添加糖 | 0（理想）|
