## 下載說明

1. **Open Access 論文**：直接點 DOI 連結，找 "Download PDF" 按鈕
2. **付費論文**：透過學校圖書館 VPN 或資料庫（IEEE Xplore、PubMed 等）下載
3. **PDF 統一放在** `docs/02_literature/papers/` 資料夾
4. **檔名格式**：`第一作者_期刊縮寫_主題_年份.pdf`
5. 下載完成後在此檔標記 ✅
