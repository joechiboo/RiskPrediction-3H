# 論文中文翻譯：使用人工智能技術的新發高血壓高精度風險預測模型

> **原文標題**：Highly precise risk prediction model for new-onset hypertension using artificial intelligence techniques
> **期刊**：The Journal of Clinical Hypertension, 2020
> **DOI**：10.1111/jch.13759

---

## 摘要

### 背景

高血壓是心血管疾病的主要風險因子。能夠預測高血壓發展風險，有助於識別高風險個體並促進早期非藥物干預，以預防未來高血壓的發生。

### 目的

本研究旨在使用機器學習方法開發一個高精度的新發高血壓風險預測模型。

### 方法

我們使用了 18,258 名日本員工的健康檢查數據（2005-2016 年），包括：
- **Year(0)**：高血壓診斷時間點
- **Year(-1)**：診斷前一年的健檢數據
- **Year(-2)**：診斷前兩年的健檢數據
- **Year(-2) 到 Year(-1) 的變化量**

數據被輸入基於機器學習方法（XGBoost 和 Ensemble）或傳統統計方法（邏輯迴歸）的模型中。

數據被隨機分為：
- **衍生集（Derivation set）**：75%（n=13,694），用於模型建構和開發
- **驗證集（Validation set）**：25%（n=4,564），用於測試模型性能

### 結果

- XGBoost 模型中最佳預測因子是 **Year(-1) 時 CAVI 測量的收縮壓**
- 驗證隊列中的 ROC 曲線下面積（AUC）：
  - **Ensemble**：0.881
  - **XGBoost**：0.877
  - **邏輯迴歸**：0.859

### 結論

我們使用機器學習方法在一般正常血壓人群中開發了一個高精度的未來高血壓預測模型。

## 引言

高血壓已成為全球嚴重的公共衛生問題，是導致心血管疾病、腎臟疾病和早逝的主要風險因子之一。預測個體未來發展為高血壓的風險，對於實施早期預防策略具有重要的臨床價值。

傳統的風險預測模型主要基於邏輯迴歸等統計方法。然而，近年來機器學習技術的發展為風險預測提供了新的可能性。機器學習方法能夠捕捉複雜的非線性關係和特徵交互作用，可能比傳統方法具有更高的預測精度。

本研究的目的是：
1. 使用機器學習方法開發新發高血壓的風險預測模型
2. 比較機器學習方法與傳統邏輯迴歸的預測性能
3. 識別高血壓發展的關鍵預測因子

---

## 方法

### 研究對象

本研究使用 2005-2016 年間的日本職場健康檢查數據。納入標準：
- 具有連續三年健康檢查記錄
- 基線時（Year(-2)）血壓正常

排除標準：
- 已有高血壓診斷
- 正在使用降壓藥物
- 數據缺失嚴重

最終納入 **18,258 名**參與者。

### 資料收集

在每次年度健康檢查中收集以下數據：

| 類別 | 變數 |
|------|------|
| **人口統計** | 年齡、性別 |
| **身體測量** | 身高、體重、BMI |
| **血壓測量** | 診所收縮壓 (SBP)、診所舒張壓 (DBP) |
| **CAVI 測量** | 心踝血管指數、CAVI 測量時的 SBP/DBP |
| **實驗室檢查** | 血糖、血脂等 |

### 特徵工程 ⭐

**核心創新：使用縱向變化量特徵**

預測模型的構建基於以下三類特徵：

1. **Year(-2) 變數**：診斷前兩年的測量值
2. **Year(-1) 變數**：診斷前一年的測量值
3. **變化量特徵**：Year(-2) 到 Year(-1) 的變化 (**Δ 特徵**)

```
預測目標：Year(0) 是否為高血壓

特徵輸入：
├── Year(-2) 測量值
├── Year(-1) 測量值
└── Δ = Year(-1) - Year(-2)  ← 變化量特徵
```

### 機器學習模型

比較三種建模方法：

| 模型 | 類型 | 說明 |
|------|------|------|
| **XGBoost** | 機器學習 | 極端梯度提升，處理非線性關係 |
| **Ensemble** | 機器學習 | 多模型集成，綜合多種算法 |
| **Logistic Regression** | 傳統統計 | 基準模型，具有可解釋性 |

### 模型訓練與驗證

- **衍生集**：75%（n=13,694）用於模型訓練
- **驗證集**：25%（n=4,564）用於性能評估

### 評估指標

- **AUC-ROC**：ROC 曲線下面積（主要指標）
- **Sensitivity**：敏感度
- **Specificity**：特異度

---

## 結果

### 基線特徵

| 項目 | 數值 |
|------|------|
| 總樣本數 | 18,258 |
| 男性比例 | 44.6% |
| 平均年齡 | 46 歲 |
| 新發高血壓病例 | 2,672 (14.6%) |
| 平均收縮壓 | 118.7 ± 11.2 mmHg |

### 模型性能比較

| 模型 | 驗證集 AUC | 排名 |
|------|-----------|------|
| **Ensemble** | **0.881** | 1 ⭐ |
| XGBoost | 0.877 | 2 |
| Logistic Regression | 0.859 | 3 |

**關鍵發現**：
- 機器學習方法（Ensemble、XGBoost）優於傳統邏輯迴歸
- AUC 提升約 2%（0.859 → 0.881）

### 特徵重要性（XGBoost 模型）

| 排名 | 特徵 | 相對重要性 |
|------|------|-----------|
| 1 | **Year(-1) CAVI 測量時的 SBP** | 100% |
| 2 | Year(-1) 診所 SBP | 57.3% |
| 3 | BMI | 重要 |
| 4 | Age | 重要 |
| 5 | CAVI | 重要 |

**重要發現**：
- **Year(-1) 的血壓測量是最重要的預測因子**
- 一年前的數據比兩年前的數據更具預測價值
- BMI 和年齡是一致的重要預測因子

## 討論

### 主要發現

1. **機器學習優於傳統統計**
   - Ensemble 和 XGBoost 的 AUC 分別達到 0.881 和 0.877
   - 比邏輯迴歸（0.859）高出約 2%
   - 這表明機器學習能夠捕捉傳統方法無法識別的複雜模式

2. **血壓測量是最重要的預測因子**
   - Year(-1) 時的收縮壓是最強的預測因子
   - 這與臨床直覺一致：當前血壓水平是未來高血壓的最佳預測指標

3. **縱向數據的價值**
   - 使用多個時間點的數據（Year(-2)、Year(-1)、變化量）提高了預測精度
   - 變化量特徵提供了額外的預測信息

### 與先前研究的比較

許多先前的預測模型將年齡和 BMI 作為主要預測因子，因為這些因素與新發高血壓密切相關。在本模型中，年齡和 BMI 也是重要的預測因子，僅次於血壓測量。

CAVI（心踝血管指數）是本研究的特色指標，它反映動脈硬化程度，已被證明與新發高血壓直接相關。

### 臨床意義

該模型可以：
1. **識別高風險個體**：在他們發展為高血壓之前
2. **促進早期介入**：實施生活方式改變等非藥物干預
3. **個性化預防**：根據個人風險水平制定預防策略

### 局限性

1. **單一種族**：僅使用日本人數據，需要在其他種族中驗證
2. **無外部驗證**：僅使用內部驗證集
3. **職場人群**：可能無法推廣到一般人群
4. **回顧性設計**：需要前瞻性研究驗證

---

## 結論

我們使用機器學習方法在一般正常血壓人群中開發了一個高精度的未來高血壓預測模型。該模型使用了三個時間點的健康檢查數據（Year(-2)、Year(-1) 和 Year(0)），包括測量值和變化量特徵。

**主要貢獻**：
1. 證明機器學習優於傳統統計方法
2. 證明縱向變化量特徵具有預測價值
3. 為高血壓早期預防提供了工具
