### 3. 新增實驗：各時間點單獨預測力比較

- 需要設計正式實驗：各時間點獨立建模，比較 AUC

### 4. 口試簡報的表達策略

- 口委不像指導教授這麼深入了解題目
- 需要講得**更完整**（背景脈絡要交代清楚）
- 但**細節少一點**（不用深入每個數字，講趨勢和結論）
- 重點：讓不熟悉的人也能跟上邏輯

### 5. 參考文獻格式：APA → IEEE

- 需捨棄目前 APA 格式（Author, Year）
