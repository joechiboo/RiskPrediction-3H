## Page 2: 研究背景

### 高血壓的重要性

- 全球**心血管疾病**的主要風險因子
- 台灣成人高血壓盛行率約 **25-30%**
- 早期發現、早期介入可有效降低併發症

### 研究缺口

| 傳統方法 | 問題 |
|----------|------|
| 單次健檢判斷 | 無法預測「未來」風險 |
| 臨床經驗法則 | 缺乏量化依據 |
| 靜態模型 | 未考慮健康指標的**變化趨勢** |

### 研究機會

> 大規模健檢資料 + 機器學習 = **個人化風險預測**

---

## Page 3: 研究目標

### 主要目標

**預測下次健檢時是否會發生高血壓**

### 創新點：虛擬健檢 (Virtual Checkup)

```
如果病人減重 5 公斤...
  → 血壓會下降多少？
  → 高血壓風險會降低多少？
```

### 臨床價值

| 傳統建議 | 本研究建議 |
|----------|-----------|
| "你要減重" | "你減 5 公斤，風險降 20%" |
| 模糊 | **量化、具體** |

### 高血壓定義

- 收縮壓 ≥ 140 mmHg，或
- 舒張壓 ≥ 90 mmHg，或
- 自我報告高血壓

## Page 5: 研究方法

### 特徵工程

| 階段 | 特徵數 |
|------|--------|
| 原始健檢特徵 | 266 個 |
| 特徵向量維度 | **1,335** |

**特徵類型**：
- 體檢數據：血壓、BMI、血液檢查
- 問卷回答：生活習慣、家族病史
- 歷史資訊：多次健檢的時序變化

### 模型選擇

| 模型 | 說明 |
|------|------|
| **XGBoost** | 梯度提升樹 |
| LightGBM | 輕量梯度提升 |
| Random Forest | 隨機森林 |

### 評估策略

- **5-fold 交叉驗證**
- 訓練集：40,488 樣本
- 測試集：74,802 樣本

## Page 6: 實驗結果

### 模型效能比較

| 模型 | AUC | Precision | Recall | F1 |
|------|-----|-----------|--------|-----|
| **XGBoost** | **88.94%** | 69.59% | 77.90% | 73.51% |
| LightGBM | 88.89% | - | - | - |
| Random Forest | 88.90% | - | - | - |
| Baseline | 較低 | - | - | - |

### 關鍵指標解讀

- **AUC 88.94%**：優秀的區分能力
- **Recall 77.90%**：找出近 8 成的高血壓患者
- **Precision 69.59%**：預測為高血壓者，約 7 成確實是

## Page 7: 虛擬健檢 (核心創新)

### 概念

> 模擬「如果病人改變某項指標」，風險會如何變化？

### 方法

1. 選擇目標：BMI 減少
2. 根據相關係數調整其他指標（如血壓）
3. 用調整後的資料重新預測
4. 比較風險變化

### 結果

| 情境 | 高風險→低風險比例 |
|------|-------------------|
| BMI 減少 1 單位 | 5.48% |
| BMI 減少 2 單位 | 6.70% |

### 臨床應用

```
醫師可以告訴病人：
"如果你減重 3 公斤（BMI -1），
 你的高血壓風險可以從 65% 降到 45%"
```

## Page 8: 關鍵發現

### 發現 1：訪問次數越多，預測越準

| 訪問次數 | 預測效能 |
|----------|----------|
| 1 次 | 較低 |
| 2-3 次 | 中等 |
| 4+ 次 | **最佳** |

> **啟示**：多時間點資料的價值

### 發現 2：前 20 個特徵就夠

- 全部 1,335 特徵 vs 前 20 特徵
- 效能**差異不大**

**Top 特徵**：
1. 收縮壓 (SBP)
2. 舒張壓 (DBP)
3. BMI
4. 年齡
5. 血糖相關指標

### 發現 3：Baseline 模型不夠

- 只用「最後一次血壓」預測 → 效能差
- 需要**多維度、多時間點**資訊

## Page 9: 研究限制與啟發

### 研究限制

| 限制 | 說明 |
|------|------|
| 資料無法公開 | MJ Health 版權限制 |
| 虛擬健檢未驗證 | 缺乏真實減重後追蹤 |
| 未用影像特徵 | 心電圖、眼底照片可能更好 |
| 計算成本 | 1,335 維特徵訓練耗時 |

### 對我們研究的啟發

| 本論文 | 我們可以學習 |
|--------|-------------|
| 多次訪問重要 | ✅ 我們用 T1, T2, T3 |
| 前 20 特徵夠 | 可做特徵選擇實驗 |
| 虛擬健檢 | 可做「虛擬 Δ」實驗 |
| AUC + F1 + Recall | 評估指標選擇 |

### 我們的優勢

| 項目 | 本論文 | 我們 |
|------|--------|------|
| 預測時間 | 下次訪問（短期） | **2-6 年（長期）** |
| 疾病數 | 僅高血壓 | **三高同時預測** |
| 可解釋性 | 特徵重要性 | **SHAP 詳細分析** |

## Page 10: 總結

### 論文貢獻

1. **大規模健檢資料應用**：20 萬人、38 萬次健檢
2. **高效能預測模型**：XGBoost AUC = 88.94%
3. **創新虛擬健檢**：量化介入效果
4. **臨床可行性**：前 20 特徵即可達到相近效能

### 核心 Take-away

> **多時間點健檢資料 + 機器學習 = 有效預測高血壓風險**
>
> **虛擬健檢 = 個人化、量化的介入建議**

### 對我們的意義

- 驗證了**多時間點**設計的價值
- 提供了**評估指標**的參考
- 啟發了**虛擬 Δ 實驗**的可能性
