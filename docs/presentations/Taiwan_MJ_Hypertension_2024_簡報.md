## Page 2: 研究背景

### 高血壓的重要性

- 全球**心血管疾病**的主要風險因子
- 台灣成人高血壓盛行率約 **25-30%**
- 早期發現、早期介入可有效降低併發症

### 研究缺口

| 傳統方法 | 問題 |
|----------|------|
| 單次健檢判斷 | 無法預測「未來」風險 |
| 臨床經驗法則 | 缺乏量化依據 |
| 靜態模型 | 未考慮健康指標的**變化趨勢** |

### 研究機會

> 大規模健檢資料 + 機器學習 = **個人化風險預測**

---

**📝 講稿（1分鐘）**

> 首先來看研究背景。高血壓是心血管疾病最重要的風險因子，在台灣成人的盛行率大約是 25% 到 30%，也就是說每三到四個成年人就有一個有高血壓。
>
> 傳統的方法有幾個問題：第一，單次健檢只能判斷「現在」有沒有高血壓，無法預測未來；第二，醫師的經驗法則缺乏量化依據；第三，過去的模型是靜態的，沒有考慮健康指標隨時間的變化趨勢。
>
> 這篇論文的研究機會就是：結合大規模健檢資料和機器學習，來做個人化的風險預測。

## Page 3: 研究目標

### 主要目標

**預測下次健檢時是否會發生高血壓**

### 創新點：虛擬健檢 (Virtual Checkup)

```
如果病人減重 5 公斤...
  → 血壓會下降多少？
  → 高血壓風險會降低多少？
```

### 臨床價值

| 傳統建議 | 本研究建議 |
|----------|-----------|
| "你要減重" | "你減 5 公斤，風險降 20%" |
| 模糊 | **量化、具體** |

---

**📝 講稿（1分鐘）**

> 這篇論文的主要目標是：預測病人在「下一次」健檢時是否會發生高血壓。注意這裡是預測「下次訪問」，是比較短期的預測。
>
> 論文最大的創新點是「虛擬健檢」的概念。什麼意思呢？就是模擬「如果病人改變某個指標，風險會怎麼變化」。比如說，如果病人減重 5 公斤，他的高血壓風險會降低多少？
>
> 這個概念的臨床價值在於：傳統上醫師只能告訴病人「你要減重」，但這篇論文可以給出量化的建議，像是「你減 5 公斤，風險可以降低 20%」，這樣的建議更具體、更有說服力。

## Page 6: 實驗結果

### 模型效能比較

| 模型 | AUC | Precision | Recall | F1 |
|------|-----|-----------|--------|-----|
| **XGBoost** | **88.94%** | 69.59% | 77.90% | 73.51% |
| LightGBM | 88.89% | - | - | - |
| Random Forest | 88.90% | - | - | - |
| Baseline | 較低 | - | - | - |

### 關鍵指標解讀

- **AUC 88.94%**：優秀的區分能力
- **Recall 77.90%**：找出近 8 成的高血壓患者
- **Precision 69.59%**：預測為高血壓者，約 7 成確實是

**📝 講稿（1分鐘）**

> 來看實驗結果。XGBoost 表現最好，AUC 達到 88.94%，這是一個相當優秀的成績。
>
> 我來解讀一下幾個重要指標：AUC 接近 0.89，代表模型有很好的區分能力，能夠有效區分會不會得高血壓的人。

> Recall 是 77.90%，意思是在所有真正會得高血壓的人當中，模型可以找出將近 8 成。這對於早期篩檢來說很重要，因為我們希望盡量不要漏掉高風險的人。

## Page 7: 虛擬健檢 (核心創新)

### 概念

> 模擬「如果病人改變某項指標」，風險會如何變化？

### 方法

1. 選擇目標：BMI 減少
2. 根據相關係數調整其他指標（如血壓）
3. 用調整後的資料重新預測
4. 比較風險變化

### 結果

| 情境 | 高風險→低風險比例 |
|------|-------------------|
| BMI 減少 1 單位 | 5.48% |
| BMI 減少 2 單位 | 6.70% |

### 臨床應用

```
醫師可以告訴病人：
"如果你減重 3 公斤（BMI -1），
 你的高血壓風險可以從 65% 降到 45%"
```

---

**📝 講稿（1分鐘）**

> 這頁是論文最核心的創新：虛擬健檢。概念是模擬「如果病人改變某個健康指標，風險會怎麼變化」。
>
> 方法是這樣的：首先選擇一個目標，比如 BMI 減少。然後根據 BMI 和其他指標的相關係數，同步調整血壓等相關指標。接著用調整後的資料重新預測，最後比較風險變化。
>
> 結果顯示：如果 BMI 減少 1 單位，大約有 5.48% 的高風險患者會變成低風險；如果 BMI 減少 2 單位，這個比例提高到 6.70%。
>
> 這個方法的臨床應用價值很高：醫師可以給病人非常具體的建議，像是「如果你減重 3 公斤，你的風險可以從 65% 降到 45%」。這樣的量化建議比「你要減重」更有說服力。

## Page 8: 關鍵發現

### 發現 1：訪問次數越多，預測越準

| 訪問次數 | 預測效能 |
|----------|----------|
| 1 次 | 較低 |
| 2-3 次 | 中等 |
| 4+ 次 | **最佳** |

> **啟示**：多時間點資料的價值

### 發現 2：前 20 個特徵就夠

- 全部 1,335 特徵 vs 前 20 特徵
- 效能**差異不大**

**Top 特徵**：
1. 收縮壓 (SBP)
2. 舒張壓 (DBP)
3. BMI
4. 年齡
5. 血糖相關指標

### 發現 3：Baseline 模型不夠

- 只用「最後一次血壓」預測 → 效能差
- 需要**多維度、多時間點**資訊

---

**📝 講稿（1分鐘）**

> 這頁我想分享三個我覺得很有意思的發現。
>
> 第一，**健檢次數越多越準**。這很直覺，但論文用數據證明了：只有一次健檢的預測效果很差，要到四次以上才最準。這讓我更確定我們用 T1、T2、T3 三個時間點是對的方向。
>
> 第二，**不需要太多特徵**。他們用了 1,335 個特徵，但其實前 20 個就夠了，像血壓、BMI、年齡這些。這對我們是好消息，因為我們只用 26 個特徵，反而更精簡、更容易解釋。
>
> 第三，**不能只看單一指標**。如果只用「上次血壓」來預測，效果很差。這說明預測疾病需要綜合多個面向的資訊，單一指標是不夠的。
