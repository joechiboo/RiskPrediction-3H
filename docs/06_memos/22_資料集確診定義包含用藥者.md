# Memo 22：資料集確診定義包含用藥者

**日期**：2026-01-29
**來源**：Zhu et al. (2024) BMJ Open, 14, e073930
**資料集 URL**：https://datadryad.org/dataset/doi%3A10.5061/dryad.z08kprrk1

## 發現

回顧資料集原始論文後，發現三高疾病的確診標記不僅基於生理指標閾值，還包含用藥與自報條件：

| 疾病 | 原始論文定義 | 之前的理解 |
|------|-------------|-----------|
| 高血壓 | SBP ≥ 140 或 DBP ≥ 90 mmHg，**或已確診且正在服用降壓藥物** | 僅數值閾值 |
| 高血糖 | FBG ≥ 7.0 mmol/L，**或自我報告糖尿病** | 僅數值閾值 |
| 高血脂 | TC ≥ 6.22 mmol/L | TC ≥ 6.2 或 TG ≥ 2.3（多寫了 TG 條件） |

## 影響

### 對模型預測的影響

用藥控制者的生理指標可能已在正常範圍內（例如服用降壓藥後血壓 120/80），但標記仍為「患病」。這造成**特徵與標籤之間的不一致**，增加預測難度：

- 模型看到正常的生理數值，卻需要預測為患病
- 這類樣本可能被模型誤判為 False Negative

### 正面意義

預測的是「確診狀態」（是否曾被診斷為三高），而非單純的「數值是否超標」。這更接近臨床實際需求 — 醫師想知道的是患者是否有三高，包含已控制者。

## 研究限制（第五章素材）

1. **確診標記混合了數值閾值與自報/用藥史**：無法區分「新發病例」與「已知且控制中的病例」
2. **資料集未提供用藥欄位**：無法進一步分析用藥對預測的影響
3. **高血脂定義較窄**：原始論文僅使用 TC ≥ 6.22，未納入 TG 標準
