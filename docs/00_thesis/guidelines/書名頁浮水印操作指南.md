## 方法一：Word 內建浮水印

### 步驟

1. 游標放在**書名頁**
2. 雙擊**頁首**區域進入編輯
3. 確認書名頁頁首「**連結到前一節**」**關閉**
4. 確認**下一節（謝辭）**頁首「連結到前一節」也**關閉**
5. **設計 → 浮水印 → 自訂浮水印 → 圖片浮水印**
6. 選取校徽圖片（watermark_ntue.jpg）
7. 勾選「**刷淡**」，縮放比例「自動」
8. 確定

### 注意事項

- 浮水印實際上是插入在**頁首**中的圖片
- 如果書名頁用了「**首頁不同**」，浮水印可能不會出現在首頁
- 若其他頁面出現浮水印：到該頁雙擊頁首 → 選取浮水印圖片 → Delete 刪除

### 已知問題

- 使用「首頁不同」隱藏頁碼時，Word 內建浮水印可能無法正確顯示在書名頁
- 轉 PDF 時浮水印可能不出現

## 方法二：手動插入圖片到頁首（備用方案）

適用於方法一無法正常顯示時：

1. 雙擊**書名頁的頁首**（會顯示「第一頁 頁首」）
2. **插入 → 圖片 → 此裝置** → 選校徽圖片
3. 選取圖片 → **文繞圖** → 「**文字在前**」（圖片在文字後方當背景）
4. 拖曳圖片到頁面**正中央**，調整大小
5. 選取圖片 → **圖片格式 → 色彩 → 刷淡**（或調低亮度/對比）

## 方法三：PDF 加浮水印（建議方式）

Word 轉 PDF 後，用 Adobe Acrobat 加浮水印最穩定：

### Adobe Acrobat Pro

1. 先用**另存新檔 → PDF** 轉檔（不要用列印轉 PDF，避免遺失書籤/目次連結）
2. 用 Acrobat 開啟 PDF
3. **工具 → 編輯 PDF → 浮水印 → 新增**
4. 來源選「**檔案**」→ 瀏覽選 `watermark_ntue.jpg`
5. 設定參數：
   - **不透明度**：100%
   - **勾選「相對目標頁面的比例」**：30%
   - **位置**：看起來在頁面之下（文字在浮水印上方）
   - **旋轉**：無
   - **垂直距離**：0 公厘，從 中心
   - **水平距離**：0 公厘，從 中心
6. 右上角「**頁面範圍選項**」→ 只選**書名頁**那一頁
7. 按**確定**

### 免費替代方案
- PDFtk
- Smallpdf 線上工具
- LibreOffice Draw 開啟 PDF 後手動加入

## PDF 保全設定

轉好 PDF 後，用 Acrobat 設定保全：

### Adobe Acrobat Pro

1. 開啟 PDF
2. **檔案 → 內容 → 安全性**（或 **保護 → 加密 → 使用密碼加密**）
3. 密碼安全性設定：

**文件開啟**
- 「要求密碼來開啟文件」→ **不勾選**（讓人可以看）

**權限**
- **勾選「限制編輯和列印文件」**
- 允許列印：**高解析度**
- 允許變更：**無**
- **不勾選**「啟用複製文字、影像和其它內容」
- **勾選**「為視力不佳者啟用螢幕閱讀程式裝置的文字協助工具」
- 設定**變更權限密碼**（自己記住！）

**選項**
- 相容性：**Acrobat X 和更高版本**
- 加密等級：**256-bit AES**
- 加密所有文件內容

4. 按**確定**
5. 系統會要求**確認權限密碼** → 再輸入一次 → 確定
6. **儲存**文件（儲存後保全才會生效）

### 注意

- 權限密碼務必記住，忘了就無法解除保全設定
- 上傳論文系統前確認 PDF 可正常開啟閱覽
- 保全設定在**儲存後**才會套用到文件
