# 書名頁浮水印操作指南

> 適用對象：國立臺北教育大學 碩士論文
> 說明：書名頁需加校徽浮水印，封面不需要

---

## 適用範圍

| 頁面 | 浮水印 | 說明 |
|------|--------|------|
| **封面** | 無 | 印在藍色銅西卡紙上 |
| **書名頁** | 校徽浮水印（刷淡） | 印在白色模造紙上 |

---

## 方法一：Word 內建浮水印

### 步驟

1. 游標放在**書名頁**
2. 雙擊**頁首**區域進入編輯
3. 確認書名頁頁首「**連結到前一節**」**關閉**
4. 確認**下一節（謝辭）**頁首「連結到前一節」也**關閉**
5. **設計 → 浮水印 → 自訂浮水印 → 圖片浮水印**
6. 選取校徽圖片（watermark_ntue.jpg）
7. 勾選「**刷淡**」，縮放比例「自動」
8. 確定

### 注意事項

- 浮水印實際上是插入在**頁首**中的圖片
- 如果書名頁用了「**首頁不同**」，浮水印可能不會出現在首頁
- 若其他頁面出現浮水印：到該頁雙擊頁首 → 選取浮水印圖片 → Delete 刪除

### 已知問題

- 使用「首頁不同」隱藏頁碼時，Word 內建浮水印可能無法正確顯示在書名頁
- 轉 PDF 時浮水印可能不出現

---

## 方法二：手動插入圖片到頁首（備用方案）

適用於方法一無法正常顯示時：

1. 雙擊**書名頁的頁首**（會顯示「第一頁 頁首」）
2. **插入 → 圖片 → 此裝置** → 選校徽圖片
3. 選取圖片 → **文繞圖** → 「**文字在前**」（圖片在文字後方當背景）
4. 拖曳圖片到頁面**正中央**，調整大小
5. 選取圖片 → **圖片格式 → 色彩 → 刷淡**（或調低亮度/對比）

---

## 方法三：PDF 加浮水印

若 Word 浮水印轉 PDF 後不顯示，可在 PDF 階段處理：

### Adobe Acrobat Pro
1. 開啟 PDF
2. **工具 → 編輯 PDF → 浮水印 → 新增**
3. 選擇「檔案」→ 選校徽圖片
4. 設定不透明度（建議 10-20%）
5. 頁面範圍：只選**書名頁**那一頁
6. 確定

### 免費替代方案
- PDFtk
- Smallpdf 線上工具
- LibreOffice Draw 開啟 PDF 後手動加入
