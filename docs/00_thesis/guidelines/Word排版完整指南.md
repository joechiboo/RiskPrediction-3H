## 1️⃣ 字型與字級設定

### 1.1 正文內容

| 語言 | 字型 | 字級 | 適用範圍 |
|------|------|------|----------|
| 中文 | **標楷體**（或新細明體） | **12號** | 所有中文內容 |
| 英文 | **Times New Roman** | **12號** | 所有英文內容 |

### 1.2 封面格式

| 項目 | 中文 | 英文 |
|------|------|------|
| 校名、系所 | 18號 楷書 | 14號 Times New Roman |
| 論文題目 | 18號 楷書 | 16號 Times New Roman |
| 撰者姓名 | 18號 楷書 | 16號 Times New Roman |
| 指導教授 | 18號 楷書 | 16號 Times New Roman |
| 年月 | 18號 楷書 | 18號 Times New Roman |

**重要提醒**：
- 行距皆為 **1.5倍行高**
- 所有行均須**置中**
- 上留白 4公分，下留白 3公分

### 1.3 書背格式

| 項目 | 字型 | 字級 |
|------|------|------|
| 校名、系所 | 標楷體 | 12號 |
| 論文題目 | 標楷體 | 12-14號 |
| 撰者姓名 | 標楷體 | - |
| 年月（數字） | 標楷體 | - |

**書背上留 2公分**

## 2️⃣ 行距設定

### 2.1 中文撰寫論文

| 內容區段 | 行距 |
|---------|------|
| **全文（含本文、摘要）** | **1.5倍行高** |
| 英文摘要 | 1.5倍行高（跟隨主文） |

### 2.2 英文撰寫論文

| 內容區段 | 行距 |
|---------|------|
| **全文（含本文、摘要）** | **2倍行高** |
| 中文摘要 | 2倍行高（跟隨主文） |

**適用原則**：
- 以論文主要撰寫語言為準
- 中文論文 → 全文 1.5倍行高
- 英文論文 → 全文 2倍行高

## 3️⃣ 頁面邊界設定

### 3.1 論文本文（最重要！）

```
上方：3 公分
下方：2 公分
左方：3 公分（方便裝訂）
右方：3 公分
```

**Word 設定路徑**：
版面配置 → 邊界 → 自訂邊界 → 輸入數值

### 3.2 封面頁

```
上方：4 公分
下方：3 公分
左方：2 公分
右方：2 公分
```

## 4️⃣ 頁碼編排

### 4.1 頁碼格式

| 區段 | 起始 | 結束 | 頁碼格式 | 字型 |
|------|------|------|----------|------|
| **書名頁、謝誌** | 書名頁 | 謝誌 | **不須編碼** | — |
| **前置頁面** | 中文摘要 | 圖目錄 | i, ii, iii... | 10號 Times New Roman |
| **正文** | 第一章 | 附錄 | 1, 2, 3... | 10號 Times New Roman |

### 4.2 頁碼位置

- **位置**：頁面**正下方**
- **對齊**：**置中對齊**

### 4.3 重要規則

**雙面印刷時（頁數 > 80頁）**：
- 各章起始頁：必須從**單數頁**開始
- 參考文獻：必須從**單數頁**開始
- 附錄：必須從**單數頁**開始

**單面印刷時（頁數 ≤ 80頁）**：
- 起始頁**不限**單數頁

### 4.4 Word 設定方式

**前置頁面（羅馬數字）**：
1. 插入 → 頁碼 → 頁面底端 → 純數字2
2. 頁碼格式 → 編號格式 → i, ii, iii...
3. 字型改為 **10號 Times New Roman**

**正文（阿拉伯數字）**：
1. 在第一章前插入**分節符號**（下一頁）
2. 取消「連結到前一節」
3. 頁碼格式 → 編號格式 → 1, 2, 3...
4. 起始頁碼 → 1
5. 字型改為 **10號 Times New Roman**

## 5️⃣ 圖表排版格式

### 5.1 表格格式

#### 表號、表名位置
- **位置**：表格**最上方**
- **對齊**：**置中**（建議）或靠左
- **格式範例**：
  ```
  表 3-1 研究架構圖
  ```

#### 表格字型
- 與論文內文一致：
  - 中文：12號 標楷體
  - 英文：12號 Times New Roman

#### 標註位置（3處）
1. **表目錄**：`表 3-1 研究架構圖 ........................... 45`
2. **論文本文**：`如表 3-1 所示...`
3. **表格本身**：表格上方的標題

#### 來源標註
- 若擷取自參考文獻，需標註來源
- 建議：**重新繪製**以避免侵權
- 格式範例：`資料來源：Lin et al. (2024)`

### 5.2 圖片格式

#### 圖號、圖名位置
- **位置**：圖片**最下方**
- **對齊**：**置中**（建議）或靠左
- **格式範例**：
  ```
  圖 4-2 實驗結果比較
  ```

#### 其他規則
- 與表格相同（標註位置、字型、來源）

### 5.3 圖表通用規則

| 規則 | 說明 |
|------|------|
| **編號順序** | 依出現順序編號（3-1, 3-2, 3-3...） |
| **位置原則** | 置於**提及該圖表的文字之後** |
| **跨頁處理** | 原則上**不跨頁**（除非超過一頁） |
| **大小限制** | 需符合版面邊界（3-2-3-3） |
| **一致性** | 同一本論文風格統一（置中/靠左） |

### 5.4 Word 製作技巧

**表格**：
1. 使用「插入 → 表格」
2. 表格上方輸入表號、表名
3. 格式 → 段落 → 置中

**圖片**：
1. 插入 → 圖片
2. 圖片下方輸入圖號、圖名
3. 版面配置 → 文繞圖 → 與文字排列

**自動編號**（進階）：
- 使用「參考資料 → 插入標號」
- 建立「表」、「圖」標號
- 可自動更新編號

## 6️⃣ 內文格式設定

### 6.1 段落設定

| 項目 | 設定值 |
|------|--------|
| **對齊方式** | **左右對齊**（Justify）⚠️ |
| **首行縮排** | 中文：空 2 字元<br>英文：空 5 格（建議） |
| **段落間距** | 與前段：0 行<br>與後段：0 行 |

**Word 設定**：
- 常用 → 段落 → 對齊方式 → **左右對齊**
- 段落 → 縮排與行距 → 特殊格式 → 首行 → 2 字元

### 6.2 標題階層

**建議層級**：最多**三層**

```
第一章  緒論                    （第一層）
  1.1 研究背景                  （第二層）
    1.1.1 三高疾病現況          （第三層）
```

**格式一致性**：
- 位置（置中/靠左）
- 大小（字級）
- 粗細（粗體/正常）
- 編號方式

### 6.3 方程式編號

**編號格式**：`(3-1)`, `(3-2)`, `(3-3)`...

**編號位置**：方程式**後方右側**

**標註方式**：
```
根據公式 (3-1) 可知...
```

**Word 設定**：
1. 插入 → 方程式
2. 使用表格排版（三欄：空白、方程式、編號）
3. 編號靠右對齊

## 7️⃣ 參考文獻格式

### 7.1 格式選擇

**建議格式**：
- **APA 格式**（最新版）
- 或依系所規定（如 IEEE）

### 7.2 排列方式（二選一）

**方式一**：按論文本文引用順序
```
[1] Lin et al. (2024)
[2] Liu et al. (2023)
[3] Wang et al. (2025)
```

**方式二**：按作者姓氏筆畫/字母順序
```
Lin et al. (2024)
Liu et al. (2023)
Wang et al. (2025)
```

### 7.3 標註位置（2處）

1. **論文本文**（引用處）：
   ```
   根據 Lin et al. (2024) 的研究顯示...
   ```

2. **參考文獻清單**：
   ```
   Lin, C.-Y., et al. (2024). Title of paper...
   ```

### 7.4 IEEE 格式範例

```
[1] Z. Wang, Y. Gu, L. Huang, et al., "Construction of machine
    learning diagnostic models for cardiovascular pan-disease based
    on blood routine and biochemical detection data," Cardiovascular
    Diabetology, vol. 23, 351, Sep. 2024.
    doi: 10.1186/s12933-024-02439-0
```

### 7.5 APA 格式範例

```
Wang, Z., Gu, Y., Huang, L., Liu, S., Chen, Q., Yang, Y., Hong, G.,
& Ning, W. (2024). Construction of machine learning diagnostic
models for cardiovascular pan-disease based on blood routine and
biochemical detection data. Cardiovascular Diabetology, 23, 351.
https://doi.org/10.1186/s12933-024-02439-0
```

## 🔟 Word 快速設定步驟

### Step 1：全域字型設定

1. **全選文件**：`Ctrl + A`
2. **設定字型**：
   - 中文：標楷體 12號
   - 英文：Times New Roman 12號
3. 快捷方式：
   - 常用 → 字型 → 標楷體、12

### Step 2：行距設定

1. **全選文件**：`Ctrl + A`
2. 段落 → 行距 → **1.5倍行高**

### Step 3：頁面邊界

1. 版面配置 → 邊界 → **自訂邊界**
2. 輸入數值：
   - 上：3 cm
   - 下：2 cm
   - 左：3 cm
   - 右：3 cm

### Step 4：段落對齊

1. **全選文件**：`Ctrl + A`
2. 常用 → 段落 → 對齊方式 → **左右對齊**

### Step 5：首行縮排

1. 選取需要縮排的段落
2. 段落 → 縮排與行距
3. 特殊格式 → **首行** → 2 字元

### Step 6：頁碼設定

**前置頁面（羅馬數字）**：
1. 插入 → 頁碼 → 頁面底端 → 純數字2
2. 頁碼格式 → i, ii, iii...
3. 字型 → 10號 Times New Roman

**正文（阿拉伯數字）**：
1. 第一章前插入分節符號（版面配置 → 分隔設定 → 下一頁）
2. 取消「連結到前一節」
3. 頁碼格式 → 1, 2, 3...
4. 起始頁碼 → 1
5. 字型 → 10號 Times New Roman

### Step 7：目錄自動生成

1. 參考資料 → 目錄 → 自訂目錄
2. 選擇格式（建議：正式）
3. 顯示階層：3

**更新目錄**：
- 右鍵目錄 → 更新功能變數 → 更新整個目錄

## 1️⃣1️⃣ 常見問題與解決

### Q1：中英文字型混合顯示不正確？

**解決方式**：
1. 全選文件（`Ctrl + A`）
2. 字型 → 進階 → 字距調整 → 不調整
3. 字型 → 中文字型：標楷體
4. 字型 → 西文字型：Times New Roman

### Q2：頁碼從特定頁開始如何設定？

**解決方式**：
1. 在起始頁前插入**分節符號**
2. 取消「連結到前一節」
3. 頁碼格式 → 起始頁碼 → 1

### Q3：圖表編號如何自動更新？

**解決方式**：
1. 使用「參考資料 → 插入標號」
2. 建立「表」、「圖」標號
3. 修改後：全選 → `F9` 更新功能變數

### Q4：雙面印刷如何確保章節從奇數頁開始？

**解決方式**：
1. 版面配置 → 分隔設定 → **奇數頁**
2. 各章節前插入分節符號（奇數頁）

### Q5：表格跨頁時標題如何重複顯示？

**解決方式**：
1. 選取表格標題列
2. 右鍵 → 表格內容 → 列
3. 勾選「**各頁頂端以標題列重複**」

## 1️⃣3️⃣ 重要提醒事項

### ⚠️ 題目一致性（7處）

論文題目必須在以下 7 處**完全一致**：
1. 封面（含書背）
2. 書名頁
3. 審定書
4. 口試成績總表
5. 口試成績個表
6. 本校學位論文授權書
7. 國圖學位論文授權書

### ⚠️ 斷行一致性

- 中/英文論文題目的斷行位置必須一致
- 封面、書名頁、審定書的斷行需相同

### ⚠️ 英文拼音

- 個人英文姓名拼音需與**護照一致**

### ⚠️ 日期填寫

- 民國/西元月份需一致（5處）：
  - 封面（3處）
  - 書名頁（2處）
- 以**繳交當時月份**填報

### ⚠️ 印出後確認

印出紙本後，務必逐一確認：
1. [ ] 頁碼是否正確（特別是目錄中的頁碼）
2. [ ] 圖表是否清晰
3. [ ] 表格是否跨頁（若跨頁是否正確）
4. [ ] 各章是否從單數頁開始（雙面印刷）
