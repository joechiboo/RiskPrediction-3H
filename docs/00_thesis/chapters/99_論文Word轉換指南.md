# 論文 Word 轉換指南

## 轉換流程

### 1. Markdown → Word 轉換（使用模板）

使用 pandoc 搭配參考模板轉換，可自動套用格式：

```bash
cd d:\Personal\Project\RiskPrediction-3H\docs\00_thesis\chapters
pandoc "01_第一章_緒論.md" -o "01_第一章_緒論.docx" --reference-doc="03_第三章_研究方法.docx"
# 其餘章節同理，替換檔名即可
```

**模板設定（首次使用需設定一次）**：

1. 開啟 `reference.docx`
2. 在 Word 中修改樣式（**常用 → 樣式 → 右鍵 → 修改**）：
   - 「內文」：標楷體 12pt、1.5 倍行高
   - 「標題 1」：標楷體 16pt 粗體
   - 「標題 2」：標楷體 14pt 粗體
   - 「標題 3」：標楷體 12pt 粗體
3. 設定頁面邊界（**版面配置 → 邊界 → 自訂邊界**）
4. 儲存模板

### 2. Word 格式調整（手動微調）

轉換完成後，在 Word 中進行以下調整：

#### 2.1 字型設定

1. **Ctrl+A** 全選
2. **常用 → 字型**：
   - 中文：標楷體 12pt
   - 英文：Times New Roman 12pt

#### 2.2 註腳字型設定

1. 找到任一註腳文字
2. 右鍵 → **樣式 → 修改「註腳文字」**
3. 設定：
   - 中文：標楷體 10pt
   - 英文：Times New Roman 10pt
   - 行距：單行間距

> 也可在 `reference.docx` 模板中預設「註腳文字」樣式，之後轉換自動套用。

#### 2.3 段落對齊：左右對齊（重要）

1. **Ctrl+A** 全選
2. **常用 → 段落 → 對齊方式 → 左右對齊**（或按 **Ctrl+J**）

> 左右對齊讓文字兩端貼齊邊界，排版更整齊正式。pandoc 預設為靠左對齊，需手動改。

#### 2.4 行距設定

1. **Ctrl+A** 全選
2. **常用 → 段落 → 行距**：1.5 倍行高

#### 2.5 頁面邊界

1. **版面配置 → 邊界 → 自訂邊界**
2. 設定：
   - 上：3.5 cm
   - 下：2.5 cm
   - 左：3 cm（裝訂側）
   - 右：2.5 cm

#### 2.6 頁碼

1. **插入 → 頁碼 → 頁面底端 → 置中**
2. 使用阿拉伯數字（1, 2, 3...）

#### 2.7 圖表獨立放置（重要）

> 參考紙本碩論風格：圖表不硬擠在段落中間，獨立呈現更正式。

**表格排版**：
1. 表格標題在上方，置中粗體（如「表 4-1 十種模型三項疾病預測結果」）
2. 表格下方接續正文說明
3. 大表（欄位多或列數多）獨佔一頁
4. 表格內字型可比正文小（10-11pt）

**圖片排版**：
1. 圖片置中放大，佔頁面寬度 80-100%
2. 圖說在下方，置中粗體（如「圖 4-1 各模型 ROC 曲線比較」）
3. 大圖獨佔一頁（如 ROC 曲線、SHAP Beeswarm、健檢次數趨勢圖）

#### 2.8 圖表標題置中

1. 找到所有「**圖 X-X**」和「**表 X-X**」標題
2. 選取後按 **Ctrl+E** 置中

#### 2.9 表格置中

1. 點擊表格內任意位置
2. **表格工具 → 版面配置 → 對齊方式 → 置中**
3. 或選取整個表格後按 **Ctrl+E**

#### 2.10 圖表不得跨頁

圖、表必須完整呈現於同一頁面，不得跨頁分割。

**Word 設定方式**：
1. 選取整個表格或圖片段落
2. **段落 → 分行與分頁設定**：
   - 勾選「與下段同頁」
   - 勾選「段落中不分頁」
3. 若表格太長無法放在同一頁，考慮：
   - 縮小字型（最小 10pt）
   - 調整欄寬
   - 拆分成兩個表格（如表 4-1a、4-1b）

## Markdown 撰寫規範

撰寫論文 Markdown 時，請遵守以下規範以確保轉換後格式正確：

### 禁止事項

1. **不要使用 `---` 分隔線**：小節之間不需要分隔線，標題本身已有層級區分
2. **不要在表格內使用粗體數值**：如 `**0.938**`，轉換後格式可能異常

### 標題格式

- 章標題：`# 第X章 標題`
- 節標題：`## X.X 標題`
- 小節標題：`### X.X.X 標題`

### 圖表標題格式

- **表格標題**（置於表格上方）：`**表 X-X 標題內容**`
- **圖片標題**（置於圖片下方）：`**圖 X-X 標題內容**`

記憶口訣：**表上圖下**

### 表格範例

```markdown
**表 3-1 研究變數定義**

| 變數 | 說明 | 單位 |
|------|------|------|
| Age | 年齡 | 歲 |
| SBP | 收縮壓 | mmHg |
```

### 圖片範例

```markdown
![研究架構圖](../figures/fig3-1_research_framework.png)

**圖 3-1 研究架構圖**
```

## 注意事項

1. **備份**：轉換前先備份現有 Word 檔（加 `_backup` 後綴）
2. **圖片路徑**：論文圖片在 `docs/00_thesis/figures/`，實驗圖片在 `docs/03_experiments/figures/`
3. **表格編號**：檢查是否有重複編號
4. **數學公式**：pandoc 轉換後可能需要手動調整
