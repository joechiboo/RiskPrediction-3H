# 論文 Word 轉換指南

## 轉換流程

### 1. Markdown → Word 轉換（使用模板）

使用 pandoc 搭配參考模板轉換，可自動套用格式：

```bash
cd d:\Personal\Project\RiskPrediction-3H\docs\00_thesis\chapters
pandoc "第三章_研究方法.md" -o "第三章_研究方法.docx" --reference-doc=reference.docx
pandoc "第四章_實驗結果.md" -o "第四章_實驗結果.docx" --reference-doc=reference.docx
```

**模板設定（首次使用需設定一次）**：

1. 開啟 `reference.docx`
2. 在 Word 中修改樣式（**常用 → 樣式 → 右鍵 → 修改**）：
   - 「內文」：標楷體 12pt、1.5 倍行高
   - 「標題 1」：標楷體 16pt 粗體
   - 「標題 2」：標楷體 14pt 粗體
   - 「標題 3」：標楷體 12pt 粗體
3. 設定頁面邊界（**版面配置 → 邊界 → 自訂邊界**）
4. 儲存模板

### 2. Word 格式調整（手動微調）

轉換完成後，在 Word 中進行以下調整：

#### 2.1 字型設定

1. **Ctrl+A** 全選
2. **常用 → 字型**：
   - 中文：標楷體 12pt
   - 英文：Times New Roman 12pt

#### 2.2 行距設定

1. **Ctrl+A** 全選
2. **常用 → 段落 → 行距**：1.5 倍行高

#### 2.3 頁面邊界

1. **版面配置 → 邊界 → 自訂邊界**
2. 設定：
   - 上：3.5 cm
   - 下：2.5 cm
   - 左：3 cm（裝訂側）
   - 右：2.5 cm

#### 2.4 頁碼

1. **插入 → 頁碼 → 頁面底端 → 置中**
2. 使用阿拉伯數字（1, 2, 3...）

#### 2.5 圖表標題置中

1. 找到所有「**圖 X-X**」和「**表 X-X**」標題
2. 選取後按 **Ctrl+E** 置中

#### 2.6 表格置中

1. 點擊表格內任意位置
2. **表格工具 → 版面配置 → 對齊方式 → 置中**
3. 或選取整個表格後按 **Ctrl+E**

## Markdown 撰寫規範

撰寫論文 Markdown 時，請遵守以下規範以確保轉換後格式正確：

### 禁止事項

1. **不要使用 `---` 分隔線**：小節之間不需要分隔線，標題本身已有層級區分
2. **不要在表格內使用粗體數值**：如 `**0.938**`，轉換後格式可能異常

### 標題格式

- 章標題：`# 第X章 標題`
- 節標題：`## X.X 標題`
- 小節標題：`### X.X.X 標題`

### 圖表標題格式

- **表格標題**（置於表格上方）：`**表 X-X 標題內容**`
- **圖片標題**（置於圖片下方）：`**圖 X-X 標題內容**`

記憶口訣：**表上圖下**

### 表格範例

```markdown
**表 3-1 研究變數定義**

| 變數 | 說明 | 單位 |
|------|------|------|
| Age | 年齡 | 歲 |
| SBP | 收縮壓 | mmHg |
```

### 圖片範例

```markdown
![研究架構圖](../figures/fig3-1_research_framework.png)

**圖 3-1 研究架構圖**
```

## 注意事項

1. **備份**：轉換前先備份現有 Word 檔（加 `_backup` 後綴）
2. **圖片路徑**：論文圖片在 `docs/00_thesis/figures/`，實驗圖片在 `docs/03_experiments/figures/`
3. **表格編號**：檢查是否有重複編號
4. **數學公式**：pandoc 轉換後可能需要手動調整
