# 三高共病關係科普

> **建立日期**：2026-01-13
> **目的**：說明高血壓、高血糖、高血脂之間的關係，作為論文背景知識

---

## 什麼是「三高」？

| 疾病 | 英文 | 診斷標準 | 影響器官 |
|------|------|----------|----------|
| **高血壓** | Hypertension (HTN) | SBP ≥ 140 或 DBP ≥ 90 mmHg | 心臟、血管、腎臟 |
| **高血糖** | Hyperglycemia (HG) | 空腹血糖 ≥ 7.0 mmol/L | 全身代謝、神經、眼睛 |
| **高血脂** | Dyslipidemia (DL) | TC ≥ 6.2 或 TG ≥ 2.3 mmol/L | 血管、心臟、肝臟 |

---

## 三高的關係：不是因果，而是共同致病路徑 + 惡性循環

### 核心概念

三高之間**不是簡單的因果關係**（A 導致 B 導致 C），而是：

1. **共同致病路徑**：三高共享相同的上游風險因子
2. **惡性循環**：一旦有了其中一種，會加速其他兩種的發展

### 共同致病路徑

```
           不良生活習慣
      (高熱量飲食、缺乏運動)
                ↓
              肥胖
                ↓
          胰島素阻抗
                ↓
    ┌──────────┼──────────┐
    ↓          ↓          ↓
  高血脂     高血壓      高血糖

    └──────────┴──────────┘
           代謝症候群
```

### 代謝症候群 (Metabolic Syndrome)

當一個人**同時**有以下 3 項或以上，稱為代謝症候群：
1. 腹部肥胖（腰圍過大）
2. 高血壓
3. 高血糖
4. 高三酸甘油酯
5. 低 HDL 膽固醇

---

## 三高之間的關係

### 1. 高血脂 與 高血壓

**常見誤解**：高血脂 → 血管堵塞 → 血流變快 → 血壓升高

**實際機制**：
```
高血脂 (尤其是 LDL)
        ↓
  沉積在血管壁
        ↓
  動脈粥狀硬化
        ↓
  血管變硬、彈性下降
        ↓
  心臟需要更大力推血 → 血壓上升
```

**時間尺度**：這個過程需要 **10-20 年**，不是立即發生

### 2. 高血壓 與 高血糖

**機制一：共同風險因子**
- 肥胖同時導致高血壓和胰島素阻抗
- 不是高血壓「造成」高血糖

**機制二：間接影響**
```
長期高血壓
    ↓
腎臟微血管受損
    ↓
腎臟調節血糖功能下降
    ↓
血糖控制變差
```

### 3. 高血糖 與 高血脂

**雙向關係**：
```
高血糖 ←→ 高血脂

高血糖 → 肝臟合成更多三酸甘油酯 → 高血脂
高血脂 → 脂肪堆積 → 胰島素阻抗 → 高血糖
```

## 為什麼三高常常一起出現？

### 「聚集現象」(Clustering)

統計上，三高的同時出現率**高於隨機預期**：

| 情況 | 若獨立 | 實際觀察 |
|------|--------|----------|
| 同時有 HTN + HG | ~1.1% | ~1.7% |
| 同時有三高 | ~0.09% | ~0.15% |

這說明三高有**共同的致病途徑**，不是獨立事件。

### 「惡性循環」

一旦有了其中一種，會**加速**其他兩種的發展：

```
        高血脂
       ↗     ↘
      ↑       ↓
 高血糖 ←───→ 高血壓
      ↓       ↑
       ↘     ↗
       代謝惡化
```

**循環機制**：

- 高血糖 → 刺激肝臟合成更多三酸甘油酯 → 高血脂
- 高血脂 → 動脈硬化、血管彈性下降 → 高血壓
- 高血壓 → 腎臟微血管受損 → 血糖調節功能下降 → 高血糖

這就是為什麼三高患者如果不積極治療，病情會越來越難控制。

## 臨床意義

### 1. 篩檢策略
- 發現一種三高 → 應同時檢查其他兩種
- 不能只治療一種，忽略其他

### 2. 預防重於治療
- 控制體重是預防三高的**最有效方法**
- 生活型態改變可同時改善三高

### 3. 風險評估
- 有一種三高的人，未來得其他三高的風險較高
- 這是本研究「多任務學習」的臨床基礎

---

## 本研究的發現

基於 SUA 資料集（13,514 樣本）的共病分析：

| 指標 | 數值 | 說明 |
|------|------|------|
| 無三高比例 | 70.6% | 大多數人是健康的 |
| HTN ↔ HG 關聯 | OR = 1.72*** | 有高血壓者，高血糖風險較高 |
| HTN ↔ DL 關聯 | OR = 1.07 (ns) | 關聯不顯著 |
| HG ↔ DL 關聯 | OR = 1.39** | 中度關聯 |
| Phi 相關係數 | 0.01 ~ 0.06 | 直接相關性弱 |

**結論**：三高之間的直接相關性較弱，但有共同的潛在風險因子。這支持使用**多任務學習 (MTL)** 同時預測三高的合理性。
