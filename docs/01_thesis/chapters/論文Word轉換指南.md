# 論文 Word 轉換指南

## 轉換流程

### 1. Markdown → Word 轉換

使用 pandoc 將各章節轉換為 Word：

```bash
cd d:\Personal\Project\RiskPrediction-3H\docs\thesis
pandoc "第三章_研究方法.md" -o "第三章_研究方法.docx" --from markdown --to docx
```

### 2. Word 格式調整

轉換完成後，在 Word 中進行以下調整：

#### 2.1 字型設定

1. **Ctrl+A** 全選
2. **常用 → 字型**：
   - 中文：標楷體 12pt
   - 英文：Times New Roman 12pt

#### 2.2 行距設定

1. **Ctrl+A** 全選
2. **常用 → 段落 → 行距**：1.5 倍行高

#### 2.3 頁面邊界

1. **版面配置 → 邊界 → 自訂邊界**
2. 設定：
   - 上：3.5 cm
   - 下：2.5 cm
   - 左：3 cm（裝訂側）
   - 右：2.5 cm

#### 2.4 頁碼

1. **插入 → 頁碼 → 頁面底端 → 置中**
2. 使用阿拉伯數字（1, 2, 3...）

#### 2.5 圖表標題置中

1. 找到所有「**圖 X-X**」和「**表 X-X**」標題
2. 選取後按 **Ctrl+E** 置中

#### 2.6 表格置中

1. 點擊表格內任意位置
2. **表格工具 → 版面配置 → 對齊方式 → 置中**
3. 或選取整個表格後按 **Ctrl+E**

## 圖片檔案命名規則

```
figures/
├── fig3-1_research_framework.png    # 圖 3-1 研究架構圖
├── fig3-1-1_timeline.png            # 圖 3-2 研究時間軸
├── checkup_distribution.png         # 圖 3-3 健檢次數分佈
└── ...
```

命名格式：`fig{章}-{編號}_{描述}.png`

---

## 注意事項

1. **備份**：轉換前先備份現有 Word 檔（加 `_backup` 後綴）
2. **圖片路徑**：確保圖片在 `figures/` 資料夾中
3. **表格編號**：檢查是否有重複編號
4. **數學公式**：pandoc 轉換後可能需要手動調整
