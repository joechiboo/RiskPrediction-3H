## 跟學弟妹分享的原則

- **不要重抄路**：能用的程式碼、文獻索引、Reference 清單，直接給。卡關靠自己長肌肉，但工具不必。
- **時序比技術重要**：論文寫不出來通常不是技術問題，是節奏問題。地圖優先講「現在該做什麼」。
- **行政流程比想像中花時間**：助教審稿、申請場地、印刷、寄口委，至少要排兩個月 buffer。
