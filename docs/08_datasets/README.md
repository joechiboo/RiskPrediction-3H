# 資料集文檔

本專案使用兩個資料集進行三高風險預測研究。

## 資料集總覽

| 資料集 | 用途 | 樣本數 | 來源 |
| ------ | ---- | ------ | ---- |
| SUA_CVDs | 主要訓練/測試 | 6,119 人 | Dryad |
| Synthea | 外部驗證 | 1,163 人 | 合成資料 |

## 詳細文檔

- [SUA_Dataset_Documentation.md](SUA_Dataset_Documentation.md) - 主要資料集（中國東南社區健檢）
- [Synthea_Dataset.md](Synthea_Dataset.md) - 外部驗證資料集（合成病患）

## 快速比較

| 特性 | SUA_CVDs | Synthea |
| ---- | -------- | ------- |
| 資料類型 | 真實健檢 | 合成資料 |
| 地區 | 中國浙江 | 美國（模擬） |
| 追蹤間隔 | 約 2 年 | 約 1 年 |
| 最少追蹤次數 | 3 次 | 3 次 |
