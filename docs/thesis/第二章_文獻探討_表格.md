## 關鍵發現對比

### Wang et al. (2024) 主要發現
1. **訪問次數越多，預測越準**
2. **前 20 個特徵就足夠**
3. **虛擬健檢可預估減重效果**

### 本研究主要發現
1. **Δ 特徵對高血壓預測貢獻顯著**
2. **class_weight 對高血糖/高血脂改善明顯**
3. **符號回歸產出可解釋公式**：
   - 高血壓：`0.106 × exp(SBP_T1)`
   - 高血糖：`0.116 × FBG_T2`
   - 高血脂：`0.052 × exp(TC_T1)`
4. **MTL 多任務學習效率提升 40%+**

## 研究定位

```
              短期預測                          長期預測
              (下次訪問)                        (2-4年後)
                  │                               │
    Wang et al.   │                               │   本研究
    ──────────────●───────────────────────────────●──────────────
                  │                               │
              單一疾病                         三種疾病
              (高血壓)                        (三高同時)
```

### 本研究的獨特貢獻
1. **長期預測**：提供 2-4 年的預警時間
2. **三高整合**：一次預測三種疾病
3. **可解釋性**：SHAP + 符號回歸雙重保障
4. **Δ 特徵**：捕捉健康變化趨勢
