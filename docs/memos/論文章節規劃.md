# 論文章節規劃

> **建立日期**：2025-12-17
> **預計總頁數**：40-80 頁

---

## 各章節比例

| 章節 | 頁數 | 佔比 | 狀態 |
|------|------|------|------|
| 第一章 緒論 | 5-8 頁 | ~10% | [ ] |
| 第二章 文獻探討 | 10-15 頁 | ~20% | [ ] |
| **第三章 研究方法** | **8-15 頁** | **~15-20%** | [ ] |
| 第四章 實驗結果 | 15-25 頁 | ~30% | [ ] |
| 第五章 結論與建議 | 5-8 頁 | ~10% | [ ] |
| 參考文獻 + 附錄 | 5-10 頁 | ~10% | [ ] |

---

## 第三章 研究方法（8-15 頁）

### 3.1 研究架構（1-2 頁）
- [ ] 整體流程圖
- [ ] 資料流向說明
- [ ] 研究步驟概述

### 3.2 資料來源與處理（2-3 頁）
- [ ] HRS 資料集介紹
- [ ] 樣本選取條件
- [ ] 特徵定義（T1, T2, T3 時間點）
- [ ] Delta 特徵工程
- [ ] 目標變數定義（高血壓、高血糖、高血脂）

### 3.3 模型方法（3-5 頁）
- [ ] Logistic Regression
- [ ] XGBoost
- [ ] 符號回歸（gplearn, PySR）
- [ ] 類別不平衡處理（class_weight）

### 3.4 評估指標（1-2 頁）
- [ ] AUC-ROC
- [ ] F1 Score
- [ ] Sensitivity (Recall)
- [ ] Specificity
- [ ] 混淆矩陣

### 3.5 實驗設計（1-2 頁）
- [ ] 訓練/測試集切分（80/20）
- [ ] 消融實驗設計
  - Delta 特徵消融
  - class_weight 消融
- [ ] 模型比較架構

---

## 寫作優先順序

### 可先寫（不會變動）
1. 3.1 研究架構
2. 3.2 資料來源與處理
3. 3.3 模型方法（數學定義）
4. 3.4 評估指標

### 等實驗穩定後再寫
1. 3.5 實驗設計（具體配置）
2. 第四章 實驗結果

---

## 備註

- 第三章主要是**方法論**，不放結果數字
- 寫清楚「怎麼做」，「做出什麼」留給第四章
- 圖表編號預留空間，方便後續插入
