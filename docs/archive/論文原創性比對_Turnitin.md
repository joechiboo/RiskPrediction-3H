## 完整作業流程

### Phase 1：申請帳號

1. 用 **Google Chrome** 開啟 https://papercheck.ntue.edu.tw/
2. 輸入帳密登入（與 **iNtue** 相同：學號 + 校務系統密碼）
3. 確認基本資料，填入要作為 Turnitin 帳號的 **Email**
4. 點選「送出申請」
5. 研發處**每週五中午 12 點前**完成當週申請的帳號建置
   - 超過週五中午 12 點申請者 → 下週五建置

### Phase 2：開通帳號

1. 收到 `noreply@turnitin.com` 寄的開通信 → 點選「**設置帳戶**」
   - 未收到請先查垃圾信件匣
2. 輸入 Email + 姓氏（無須全名） → 點「下一步」
3. 再收一封密碼創建信 → 點「**創建密碼**」（連結 24 小時內有效）
4. 設定密碼（至少 8 個字符）
5. 登入 → 設定使用者資訊 → 點「我同意-繼續」

### Phase 3：上傳比對

1. 登入 Turnitin（https://www.turnitin.com/login_page.asp?lang=zh_tw）
2. 點選「**課程**」
3. 選擇當年度其中一份**作業匣** → 點「開啟」
4. 點選「**上傳提交**」
5. 輸入**論文名稱**（提交標題） + 選擇檔案上傳
6. 確認內容 → 點「交至 Turnitin」

### Phase 4：等待 + 檢視報告

1. 上傳成功後自動跳回作業匣，等待比對結果
   - 前 3 次：**1-2 小時**出結果
   - 第 4 次起：每次等 **24 小時**
2. 「相似度」出現百分比 → 點進去檢視比對報告
3. 右上角數字 = 相似比例，可看細部來源

### Phase 5：下載報告

點選「**下載**」，分別下載：

1. **「目前檢視」** — 有百分比的原創性報告 PDF
2. **「電子回條」** — 提交回條

### Phase 6：簽名 + 繳交

1. 印出上述 **2 份紙本**
2. 學生本人**簽名 + 日期**
3. 指導教授**簽名 + 日期**
4. 繳交給系所助教 → 啟動學位口試申請

## 比對次數規則

| 次數 | 出結果時間 |
|:---:|------|
| 第 1-3 次 | **1-2 小時**（同一作業匣內） |
| 第 4 次起 | 每次等 **24 小時** |

- 在學期間可**無限次**比對

### 通過標準（國北教大要點）

| 條件 | 標準 |
|------|------|
| 總相似度 | **≤ 20%** |
| 單一來源相似度 | **≤ 2%** |

> 系上未另訂標準，先以校級要點為準。

---

## 比對策略

| 次數 | 用途 | 上傳檔案 | 預計時間 |
|:---:|------|------|------|
| 第 1 次 | 初次比對，了解相似度分布 | 初稿III.pdf | 帳號開通後立即 |
| 第 2 次 | 根據結果修改後再比對 | 修改後版本 | 視修改進度 |
| 第 3 次 | 最終版（拿去簽名繳交） | 定稿 PDF | 口試申請前 |

> 不要浪費前 3 次快速額度在「測試上傳」，直接用論文跑。

## 時程規劃

```
3/16（日）：申請帳號（papercheck.ntue.edu.tw）
3/21（五）：帳號建置完成 → 收開通信 → 設密碼
3/21 當天：上傳初稿III.pdf → 等 1-2 小時出結果
     ↓
看報告 → 和指導教授討論
     ↓
如需修改 → 改完跑第 2 次 → 確認 OK → 跑第 3 次定稿版
     ↓
印出 2 份 → 雙方簽名 → 繳交助教
```

- **建議時程**：第二學期畢業者，4 月初前完成
