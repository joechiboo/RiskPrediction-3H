# 論文章節規劃

> **建立日期**：2025-12-17
> **更新日期**：2026-01-08
> **預計總頁數**：40-80 頁
> **結構依據**：教授 Meeting 指導

---

## 教授建議的內容順序

```
1. Introduction
2. Related work + Motivation + Problem definition
3. 方法
4. 結果 (Experimentation, Discussion)
5. 結論/建議
6. 附錄
```

---

## 各章節比例（6 章結構）

| 章節 | 內容 | 頁數 | 佔比 | 狀態 |
|------|------|------|------|------|
| 第一章 | 緒論 (Introduction) | 5-8 頁 | ~10% | [ ] |
| 第二章 | 文獻探討與問題定義 | 12-18 頁 | ~25% | [ ] |
| 第三章 | 研究方法 | 8-15 頁 | ~20% | [ ] |
| 第四章 | 實驗結果 | 12-18 頁 | ~25% | [ ] |
| 第五章 | 討論 | 6-10 頁 | ~15% | [ ] |
| 第六章 | 結論與建議 | 5-8 頁 | ~10% | [ ] |
| 附錄 | 參考文獻 + 補充資料 | 5-10 頁 | - | [ ] |

## 備註

- 第三章主要是**方法論**，不放結果數字
- 第四章合併結果與討論，更緊湊
- Problem Definition 放在第二章末，承接文獻、引出方法
- 圖表編號預留空間，方便後續插入
