# 資料集探索歷程與未來研究方向

## 備忘錄摘要

記錄本研究在資料集選擇過程中的探索與挑戰，以及未來可行的研究方向。

---

## 一、風險預測模型的資料需求

### 必要條件
```
理想資料集 = 縱貫追蹤 + 血液檢驗
             ↓           ↓
        (同一人多年)   (客觀診斷)
```

| 需求 | 說明 |
|------|------|
| 縱貫追蹤 | 同一個體多次測量，可觀察 T0 → T1 → T2 的變化 |
| 血液檢驗 | 空腹血糖、血脂、肝腎功能等客觀生化指標 |

---

## 二、本研究採用的資料集

### SUA Dataset（主要資料集）
- **來源**：台灣健康檢查資料
- **期間**：2010-2018，每兩年一次
- **樣本**：25,744 筆記錄
- **優勢**：
  - ✅ 縱貫追蹤（同一人 Times 1-4）
  - ✅ 完整血液檢驗
  - ✅ 適合風險預測研究

---

## 三、嘗試過但未採用的資料集

### 1. HRS (Health and Retirement Study)

| 項目 | 說明 |
|------|------|
| **嘗試原因** | 大規模美國縱向研究，資料公開 |
| **優點** | 豐富人口學資料、疾病診斷史、藥物使用 |
| **致命限制** | 缺乏血液檢驗資料（無血糖、血脂、肝腎功能） |
| **結論** | 無法滿足三高風險預測的核心需求 |

### 2. NHANES (National Health and Nutrition Examination Survey)

| 項目 | 說明 |
|------|------|
| **嘗試原因** | 完整血液檢測、資料品質高 |
| **優點** | 標準化生化檢驗、代表性佳 |
| **致命限制** | 純橫斷面設計，每次調查為不同受試者 |
| **結論** | 無法追蹤個體變化，不適合風險預測 |

### 3. UK Biobank

| 項目 | 說明 |
|------|------|
| **優點** | 50萬人規模、完整生化與基因資料 |
| **限制** | 需申請、以英國人群為主 |
| **狀態** | 未申請，考量人群差異 |

### 4. Framingham Heart Study

| 項目 | 說明 |
|------|------|
| **優點** | 最經典心血管縱貫研究、三代追蹤 |
| **限制** | 需申請、不完全公開 |
| **狀態** | 未申請 |

## 四、未來研究方向

### 台灣本土資料驗證

#### 1. 美兆健康資料庫 (MJ Health Database)
| 項目 | 說明 |
|------|------|
| **管理單位** | 財團法人演譯基金會 |
| **資料內容** | 健康問卷 + 生化檢驗（100+ 項目） |
| **特殊優勢** | 涵蓋健康及亞健康族群（非僅病患） |
| **追蹤期間** | 1996 年起 |
| **申請方式** | 提交申請案 → 繳費 → 取得授權 |
| **聯絡方式** | contact_us@mjhrf.org |
| **適用性** | 非常適合預防醫學研究 |

#### 2. 台灣人體生物資料庫 (Taiwan Biobank)
| 項目 | 說明 |
|------|------|
| **規模** | 目標 20 萬人 |
| **內容** | 基因 + 環境因子 + 健康追蹤 |
| **適用性** | 可進行外部驗證 |

#### 3. 健保資料庫
| 項目 | 說明 |
|------|------|
| **優勢** | 涵蓋全人口、完整就醫紀錄 |
| **限制** | 無生化檢驗值 |
| **適用性** | 可串聯追蹤實際發病結果 |

### 國際資料驗證

- 可申請 UK Biobank 進行跨人群驗證
- 驗證模型在不同種族/地區的泛化能力

## 五、研究限制

### 本研究的限制

1. **資料來源單一**
   - 主要使用 SUA Dataset
   - 未進行外部資料集驗證

2. **地域限制**
   - 資料來源為特定區域健檢中心
   - 結果推廣至其他族群需謹慎

3. **資料可及性**
   - 部分理想資料集（如 Framingham、UK Biobank）需申請
   - 美兆資料庫需付費取得

### 無法克服的挑戰

| 資料集 | 挑戰 |
|--------|------|
| NHANES | 橫斷面設計，根本無法用於風險預測 |
| HRS | 缺乏生化檢驗，無法替代 |
| 健保資料庫 | 有就醫紀錄但無檢驗值 |

## 六、建議未來研究

1. **申請美兆資料**進行台灣本土驗證
2. **串聯健保資料庫**追蹤長期發病結果
3. **申請 Taiwan Biobank**進行基因與環境交互作用分析
4. **跨國驗證**以評估模型泛化能力

---

## 七、經驗總結

> 「資料集的適用性評估應在研究設計初期進行，避免後期發現關鍵變數缺失。」

### 關鍵學習

1. **風險預測研究**必須使用縱貫資料
2. **三高研究**必須有血液檢驗數據
3. **橫斷面資料**（如 NHANES）雖然品質高，但不適合預測模型
4. **完美資料集不存在**，需權衡各項條件
