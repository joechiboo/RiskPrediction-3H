## 1. 引言（Introduction）

### 1.1 全球糖尿病負擔

第二型糖尿病（T2D）的全球盛行率持續上升，2021年估計有5.29-5.37億成人受影響，預計到2050年將達到13.1億人[1,2]。

在台灣，NAHSIT 2013-2016資料顯示，近三分之一的成人有空腹血糖異常[3]，然而僅有4.1%的第二型糖尿病患者達到美國糖尿病協會（ADA）對於糖化血紅素、血壓和膽固醇的控制目標[4]。

### 1.2 糖尿病年輕化趨勢

糖尿病發病年齡逐漸下降[5]。由於疾病持續時間與併發症發生率高度相關，年輕患者在其一生中預期會經歷更多併發症[6]。因此，美國糖尿病協會建議從35歲開始進行糖尿病篩檢[7]。

### 1.3 文獻回顧與研究缺口

統計學和機器學習方法已被用於預測中國、韓國及其他人群的糖尿病或前驅糖尿病[8-12]，但大多數研究為橫斷面研究或針對老年人群[8-10]，針對年輕成人的研究很少[12]。此外，像**Shapley加法解釋（SHAP）**這樣的可解釋性方法很少被應用[13,14]，且先前的模型通常只針對單一結果（連續變化或二元分類）。

### 1.4 本研究創新點

相比之下，我們的研究利用了一個大型縱向世代的台灣年輕男性，平均追蹤時間接近6年，採用了**雙機器學習（ML）框架**（連續空腹血糖變化δ-FPG和二元前驅糖尿病分類），整合了**SHAP進行可解釋性分析**，並通過重複運行和信賴區間確保可重複性。這些優勢使我們的模型更具臨床可操作性，用於早期前驅糖尿病風險分層。

近期評論進一步支持使用SHAP和其他可解釋AI（XAI）工具來實現透明的臨床決策支持[15,16]。

### 1.5 二元ML分類模型的重要性

這種可解釋性在**二元ML分類模型**中特別有價值，這些模型預測分類結果，如前驅糖尿病的存在與否。與追蹤漸進代謝變化的連續模型不同，**二元方法直接與臨床實踐中使用的診斷閾值對齊**[17-19]，使其特別適合篩檢計畫和早期干預規劃。

當與SHAP結合時，二元模型不僅提供高預測準確性，還能清楚識別驅動分類決策的最具影響力因素。這種**雙重能力——準確預測和透明推理**——使二元ML模型成為連續預測框架的重要補充，確保統計穩健性和臨床適用性。

### 1.6 研究目標

在本研究中，我們招募了**6247名台灣年輕男性**，追蹤時間為**5.9年**。通過使用四種不同的ML方法，我們的目標如下：

1. **比較多元邏輯迴歸（MLR）和ML方法的準確性**
2. **識別δ-FPG最重要的風險因子**（定義為追蹤結束時FPG減去基線FPG）
3. **使用SHAP進一步檢查每個因子的影響和方向**

除了將δ-FPG建模為連續變量外，我們還旨在**開發和評估一個分類（二元）ML模型**，根據人口統計、生化和生活方式因素將個體分類為正常或前驅糖尿病，從而使預測建模與臨床相關的診斷閾值對齊。

## 2. 材料與方法（Materials and Methods）

### 2.1 參與者與研究設計

本研究使用的方法已在我們先前的出版物中描述[11]。資料來自**台灣MJ世代**，這是一個基於MJ健康篩檢中心進行的健康檢查的持續前瞻性世代[20]。

這些檢查包括100多項生物指標，如人體測量、血液檢查和影像學檢查。每位參與者還完成了一份自填問卷，涵蓋個人和家族病史、當前健康狀況、生活方式、身體活動、睡眠習慣和飲食模式[20]。

本研究為**次級資料分析**。在健康檢查時，參與者提供了未來匿名研究使用的一般同意。資料庫由MJ Health研究基金會維護。本研究使用的全部或部分資料由MJ Health研究基金會授權和提供（授權碼：**MJHRF2023007A**）。本文提出的解釋和結論不一定代表基金會的觀點。有關更多詳細資訊，請參閱基金會的年度技術報告[21]。

研究方案獲得**高雄國軍總醫院機構審查委員會**批准（IRB編號：**KAFGHIRB 112-006**，批准日期：2023年6月21日）。由於未從患者處收集樣本，IRB批准了簡短審查，無需同意書。

### 2.3 兩個模型的建立

從我們的初步評估中，**模型1包括所有25個變量**。我們的結果顯示，**FPGbase的重要性為100%**，相比第二重要因子（BF，17.6%）。為了進一步評估這些因子之間的隱藏交互作用，通過**移除FPGbase建立了模型2**。

## 2.3 機器學習方法

我們採用了**兩個互補的ML框架**來預測前驅糖尿病風險並從人口統計、生化和生活方式資料中識別關鍵預測因子：

- **第一個框架**：將結果建模為**連續變量——δ-FPG**
- **第二個框架**：將結果建模為**二元結果（正常 vs. 前驅糖尿病）**

這種雙重設計使我們能夠捕捉**細緻的代謝變異**和**臨床可操作的閾值**。

### 2.3.1 連續結果預測（δ-FPG）

訓練了四種ML演算法來預測δ-FPG（**圖2**）：

1. **隨機森林（RF）**[22]
2. **隨機梯度提升（SGB）**[23]
3. **極端梯度提升（XGBoost）**[24]
4. **彈性網路（EN）**[25]

#### 數據分割與超參數調整

- 資料按**80/20分割為訓練集和測試集**
- 通過**10折交叉驗證**進行**網格搜索**調整超參數
- 使用以下指標評估性能（**表3**）：
  - **SMAPE**：對稱平均絕對百分比誤差
  - **RAE**：相對絕對誤差
  - **RRSE**：相對平方根誤差
  - **RMSE**：均方根誤差

```

                    ↓

                    ↓

```

#### 表3. 連續結果預測模型的性能指標公式

| 指標 | 描述 | 計算公式 |
|------|------|----------|
| **SMAPE** | 對稱平均絕對百分比誤差 | SMAPE = (1/n) Σ \|yi - ŷi\| / ((\|yi\| + \|ŷi\|)/2) × 100 |
| **RAE** | 相對絕對誤差 | RAE = √[Σ(yi-ŷi)² / Σ(yi-ȳi)²] |
| **RRSE** | 相對平方根誤差 | RRSE = √[Σ(yi-ŷi)² / Σ(yi-ȳi)²] |
| **RMSE** | 均方根誤差 | RMSE = √[(1/n) Σ(yi - ŷi)²] |

其中 ŷi 和 yi 分別表示預測值和實際值；n 代表實例數。

#### 穩健性與可重複性

為確保穩健性，**訓練/測試過程重複10次**，使用不同的隨機種子。**表6**呈現了每個指標的平均值和95%信賴區間（CI）。區間極窄（通常上下界相同），表明模型性能在運行中高度穩定和可重複。

例如，RF的RMSE一致為：
- **平均值 = 6.471158**
- **SE = 0.00179**
- **95% CI = 6.467109–6.475208**

特徵重要性分數在所有運行和模型中取平均值。使用來自**20次重複訓練-測試循環**（不同隨機種子）的SHAP衍生重要性的箱線圖檢查特徵重要性分佈：

- **窄箱線圖**：表示跨運行穩定的貢獻
- **寬分佈**：突出顯示不穩定或上下文依賴重要性的特徵

此程序已被推薦用於評估機器學習中解釋方法的可靠性[26]。

#### 敏感性分析

由於FPGbase的主導影響（初始模型中重要性為100%），進行了**計劃的敏感性分析**以移除FPGbase。相同的建模過程應用於**模型2（無FPGbase）**以識別替代預測因子。

模型1和模型2的特徵重要性均呈現在**第3.3節**。

#### SHAP可解釋性分析

使用應用於**隨機森林演算法**的**SHAP**進一步探索**FPGbase排除模型（模型2）**的模型可解釋性：

- **SHAP蜂群圖**：可視化每個特徵對個體預測的影響方向和大小
- **平均絕對SHAP值**：提供特徵重要性的全局排名
- **淨SHAP值**：表明特徵是否發揮主要正面或負面影響

關鍵特徵的依賴圖（例如，追蹤間隔和基線FPG）也顯示在**第3.3節**，說明特徵效應如何被其他變量調節。

#### 殘差診斷

連續模型的殘差診斷，包括**分位數-分位數圖**和**殘差 vs. 擬合值**，呈現在**圖3**，確認對模型假設的合理遵守。

## 2.3.2 二元結果預測（正常 vs. 前驅糖尿病）

### 模型開發

對於分類，我們開發了一個**基於XGBoost的流程**[24]，使用**70/30衍生/驗證分割**：

#### 特徵工程

- 臨床相關比率：**TG/HDL-C、LDL-C/HDL-C、腰臀比**
- 對數轉換：**TG、γ-GT、ALP**

#### 資料處理

- **缺失值**：使用**中位數值插補**[27]
- **類別不平衡**：使用**SMOTE-Tomek**[28,29]處理
- 我們還考慮了不平衡補救措施的近期證據和評論，以減輕過度樂觀和校準錯誤風險[30,31]

#### 特徵選擇

- 使用來自基線XGBoost模型的**排列重要性**
- 保留**前20個預測因子**

#### 超參數調整

- 使用**Optuna**[32]在**三折分層交叉驗證**中進行調整
- **100次試驗**
- **20輪後早停**
- **固定隨機種子（42）**以確保可重複性

#### 性能評估

在驗證集上使用以下指標評估最終模型性能：

- **準確度（Accuracy）**
- **精確度（Precision）**
- **召回率（Recall）**
- **特異度（Specificity）**
- **F1分數**
- **ROC-AUC**
- **PR-AUC**

#### 可解釋性

使用**SHAP**[13]評估模型可解釋性。

在**閾值0.50**處顯示混淆矩陣。鑑於類別不平衡，**精確度-召回曲線**被優先用於傳達檢索性能[33]。

### 防止資料洩漏

為防止資料洩漏，**所有預處理步驟**——包括缺失值插補、特徵工程、SMOTE-Tomek重採樣和特徵選擇——在交叉驗證期間**獨立應用於每個訓練折**：

- **插補的中位數**：僅從訓練折計算，並應用於訓練和驗證子集
- **SMOTE-Tomek**：僅應用於訓練折，不納入任何驗證資料
- **基於排列重要性的特徵選擇**：使用僅在訓練折上訓練的模型執行

這些選擇符合BMJ關於評估臨床預測模型的最新指南，涵蓋**區分度、校準和外部驗證**[34,35]。

### 模型校準評估

使用以下方法評估模型校準：

#### Brier分數

定義為：

```
Brier Score = (1/n) Σ(p̂i - yi)²
```

其中：
- **p̂i**：第i個個體前驅糖尿病的預測機率
- **yi**：觀察到的結果（如果是前驅糖尿病則為1，否則為0）
- **n**：驗證集中的個體數

較低的Brier分數表示更好的校準，**0表示完美校準**。

#### 可靠性（校準）曲線

將預測機率與跨**10個預測風險區間**的觀察結果進行比較，提供校準的視覺評估。

校準圖（**圖14**）顯示預測風險在各區間與觀察結果良好對齊。它表明模型的預測機率與觀察結果密切對齊，特別是在**較高風險水平（>0.6）**，儘管在較低機率處觀察到輕微的低估。

### 臨床實用性與閾值分析

為支持篩檢環境中的臨床實用性，我們評估了兩個操作閾值：

#### 1. 高敏感度閾值（0.2892）

- **目的**：優先最小化假陰性
- **性能**（**表10**）：
  - **敏感度**：99.53%
  - **特異度**：47.46%
- **適用場景**：人群級篩檢，即使增加假陽性，識別所有高風險個體也至關重要

#### 2. 平衡閾值（0.5683）

- **確定方法**：使用**Youden's J統計量**
- **目的**：最大化敏感度和特異度之和
- **性能**（**表11**）：
  - **Youden's J**：0.7930
  - **敏感度**：88.69%
  - **特異度**：90.61%
- **適用場景**：確認性評估或針對性干預計畫

Youden's J定義為：

```
Youden's J = (敏感度 + 特異度 − 1)
```

### 決策曲線分析（DCA）

進行**DCA**以評估二元分類器的臨床實用性，通過量化使用模型指導臨床決策的**淨效益**，跨一系列閾值機率[36,37]。

淨效益計算為**真陽性比例與加權假陽性比例之間的差異**，其中權重為**閾值機率 / (1-閾值機率)**。

這允許將模型的性能與兩種極端策略進行比較：**治療所有患者或不治療任何患者**。

DCA使用來自驗證集的預測機率和真實結果標籤實施，跨閾值機率範圍從**0.01到0.99**。

如**圖15**所示，RF模型在廣泛的閾值機率範圍（10%至80%）內提供比「治療所有」和「不治療」策略**更大的淨效益**。這表明使用模型指導篩檢和干預決策將比普遍篩檢或不篩檢帶來更好的患者結果。

模型的臨床實用性在**中等風險範圍內最高**，支持其作為年輕男性針對性預防工具的使用。

## 2.3.4 信賴區間估計

為量化模型性能的不確定性，我們根據每個模型**10次重複訓練-測試循環**獲得的分數分佈計算每個評估指標（SMAPE、RMSE、RAE、RRSE）的**95% CI**。

對於每個指標，計算平均值和標準誤差（SE），CI估計為：

```
CIs = x̄ ± tα/2, n−1 × SE
```

其中：
- **x̄**：指標的樣本平均值
- **tα/2, n−1**：來自自由度為n−1的Student's t分佈的臨界值[38]
- **SE**：平均值的標準誤差

這種方法遵循既定的小樣本推斷統計實踐[39]，並且當指標的抽樣分佈近似正態時適用[40]。
