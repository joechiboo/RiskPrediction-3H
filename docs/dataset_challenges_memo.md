# 📝 資料集挑戰備忘錄

## 🔍 大部分資料集的共同問題

### 1. **缺乏相同個案累積多年的紀錄**
- 多數公開資料集是**橫斷面調查**（cross-sectional）
- 每次調查是不同受試者，無法追蹤個體變化
- **問題**：無法建立風險預測模型（不知道誰後來發病）
- **例子**：NHANES、KNHANES、大部分國家健康調查

### 2. **缺乏血液檢驗的紀錄**
- 許多縱貫性資料集只有**問卷調查**或**自我報告**
- 缺少客觀的生化檢驗數據（血糖、血脂、肝腎功能等）
- **問題**：無法準確診斷三高疾病
- **例子**：許多社區調查、電話訪問調查
